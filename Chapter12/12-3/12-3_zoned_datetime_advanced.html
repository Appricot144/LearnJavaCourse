ZonedDateTime：タイムゾーン付きの日時

LocalDateTime にタイムゾーン情報を加えたクラスです。
国際的なシステムや「UTC からの差分」が必要な場面で使います。

▶ 同じ瞬間でもタイムゾーンによって表示が異なる
Asia/Tokyo（UTC+9）
→
2024-03-15T12:00+09:00
Europe/London（UTC+0）
→
2024-03-15T03:00Z
America/New_York（UTC-5）
→
2024-03-14T22:00-05:00
ZonedDateTime の使い方
import java.time.*;

// 現在時刻をタイムゾーン付きで取得
ZoneId    tokyo   = ZoneId.of("Asia/Tokyo");
ZoneId    london  = ZoneId.of("Europe/London");
ZoneId    ny      = ZoneId.of("America/New_York");

// LocalDateTime にタイムゾーンを付与
LocalDateTime ldt  = LocalDateTime.of(2024, 3, 15, 12, 0);
ZonedDateTime zdtT = ldt.atZone(tokyo);

System.out.println(zdtT);
// 2024-03-15T12:00+09:00[Asia/Tokyo]

// 他のタイムゾーンに変換
ZonedDateTime zdtL = zdtT.withZoneSameInstant(london);
ZonedDateTime zdtN = zdtT.withZoneSameInstant(ny);

System.out.println(zdtL);  // 2024-03-15T03:00Z[Europe/London]
System.out.println(zdtN);  // 2024-03-14T22:00-05:00[America/New_York]

// 時刻だけ取り出す
System.out.println("NY時刻: " + zdtN.toLocalTime());  // 22:00

ChronoUnit：期間の計算

2つの日付・時刻の差を特定の単位で取得します。

ChronoUnit の使い方
import java.time.*;
import java.time.temporal.ChronoUnit;

LocalDate d1 = LocalDate.of(2020, 1, 1);
LocalDate d2 = LocalDate.of(2024, 3, 15);

long years  = ChronoUnit.YEARS.between(d1, d2);
long months = ChronoUnit.MONTHS.between(d1, d2);
long days   = ChronoUnit.DAYS.between(d1, d2);
long weeks  = ChronoUnit.WEEKS.between(d1, d2);

System.out.println(years  + "年");    // 4年
System.out.println(months + "ヶ月");  // 50ヶ月
System.out.println(days   + "日");    // 1535日
System.out.println(weeks  + "週間");  // 219週間

TemporalAdjusters：便利な日付調整

TemporalAdjusters の使い方
import java.time.*;
import java.time.temporal.TemporalAdjusters;

LocalDate date = LocalDate.of(2024, 3, 15);

// 月初・月末
LocalDate firstDay = date.with(TemporalAdjusters.firstDayOfMonth());
LocalDate lastDay  = date.with(TemporalAdjusters.lastDayOfMonth());
System.out.println(firstDay);  // 2024-03-01
System.out.println(lastDay);   // 2024-03-31

// 次の月曜日
LocalDate nextMon = date.with(TemporalAdjusters.next(DayOfWeek.MONDAY));
System.out.println(nextMon);   // 2024-03-18

// 年初・年末
LocalDate firstOfYear = date.with(TemporalAdjusters.firstDayOfYear());
LocalDate lastOfYear  = date.with(TemporalAdjusters.lastDayOfYear());
System.out.println(firstOfYear);  // 2024-01-01
System.out.println(lastOfYear);   // 2024-12-31

Stream と Date/Time API の組み合わせ

日付リストの Stream 処理
import java.time.*;
import java.util.*;
import java.util.stream.*;

List<LocalDate> dates = List.of(
    LocalDate.of(2024, 1, 15),
    LocalDate.of(2024, 3, 20),
    LocalDate.of(2024, 2, 10),
    LocalDate.of(2024, 5, 5),
    LocalDate.of(2024, 4, 1)
);

// 3月以降の日付を昇順で出力
dates.stream()
    .filter(d -> d.getMonthValue() >= 3)
    .sorted()
    .forEach(System.out::println);
// 2024-03-20
// 2024-04-01
// 2024-05-05

// 月ごとにグループ化
Map<Month, List<LocalDate>> byMonth = dates.stream()
    .collect(Collectors.groupingBy(LocalDate::getMonth));
System.out.println(byMonth);

Date/Time API クラス一覧

クラス 用途
LocalDate 日付のみ（タイムゾーンなし）
LocalTime 時刻のみ（タイムゾーンなし）
LocalDateTime 日付＋時刻（タイムゾーンなし）
ZonedDateTime 日付＋時刻＋タイムゾーン
Period 年・月・日 単位の期間
Duration 時・分・秒 単位の期間
DateTimeFormatter 書式変換（format / parse）
ChronoUnit 特定単位での差の計算
TemporalAdjusters 月初・月末・次の曜日など

コード例：営業日の計算

BusinessDays.java
import java.time.*;
import java.util.stream.*;

public class BusinessDays {

    // 営業日かどうか（土日以外）
    static boolean isBusinessDay(LocalDate date) {
        DayOfWeek dow = date.getDayOfWeek();
        return dow != DayOfWeek.SATURDAY && dow != DayOfWeek.SUNDAY;
    }

    // 指定期間内の営業日数を数える
    static long countBusinessDays(LocalDate start, LocalDate end) {
        return start.datesUntil(end.plusDays(1))
            .filter(BusinessDays::isBusinessDay)
            .count();
    }

    public static void main(String[] args) {
        LocalDate start = LocalDate.of(2024, 1, 1);
        LocalDate end   = LocalDate.of(2024, 1, 31);

        System.out.println("2024年1月の営業日数: "
            + countBusinessDays(start, end));

        // 次の営業日
        LocalDate today = LocalDate.of(2024, 3, 15);  // 金曜日
        LocalDate next  = today.plusDays(1);
        while (!isBusinessDay(next)) next = next.plusDays(1);
        System.out.println("次の営業日: " + next);
    }
}
▶ 出力結果
2024年1月の営業日数: 23
次の営業日: 2024-03-18

解説
  • YEARS.between(2023-04-01, 2024-03-31)：1年未満のため 0。
  • MONTHS.between：11ヶ月。
  • DAYS.between：365日（2024年はうるう年だが期間に2/29が含まれるかで変わる。4/1〜3/31 = 365日）。
  • 2024-06-15 は土曜日。next(MONDAY) → 6月17日（月）。nextOrSame(SATURDAY) → 今日自体が土曜日なのでそのまま 6月15日。

import java.time.*;
import java.util.*;
import java.util.stream.*;

public class Q2 {
    public static void main(String[] args) {
        List<LocalDate> dates = List.of(
            LocalDate.of(2024, 3, 4),   // 月
            LocalDate.of(2024, 3, 9),   // 土
            LocalDate.of(2024, 3, 10),  // 日
            LocalDate.of(2024, 3, 15),  // 金
            LocalDate.of(2024, 3, 18)   // 月
        );

        // 平日だけフィルタ
        dates.stream()
            .filter(d -> d.getDayOfWeek() != DayOfWeek.SATURDAY
                      && d.getDayOfWeek() != DayOfWeek.SUNDAY)
            .forEach(d -> System.out.println(
                d + " (" + d.getDayOfWeek() + ")"));

        System.out.println("---");

        // 月ごとの件数（全て3月なので1グループ）
        dates.stream()
            .collect(Collectors.groupingBy(
                LocalDate::getMonth, Collectors.counting()))
            .forEach((m, c) -> System.out.println(m + ": " + c + "件"));

        System.out.println("---");

        // 最も遅い日付
        dates.stream()
            .max(Comparator.naturalOrder())
            .ifPresent(System.out::println);
    }
}

問題 12-3-3
コード記述 ⭐⭐⭐

次の条件を満たす EventPlanner クラスを作成してください。

  • 基準日を 2024-04-01 とする
  • 以下の処理をすべて行い出力する
処理 内容
月末の営業日 4月の月末（4/30）が平日かどうか確認し、土日なら前の金曜日を返す
四半期末 第1四半期（1〜3月）の末日（TemporalAdjusters使用）
残り営業日数 基準日から月末まで（4/1〜4/30）の平日の数を datesUntil で数える
次の月曜日 基準日以降の最初の月曜日（nextOrSame）
▶ 期待される出力
月末営業日: 2024-04-30
第1四半期末: 2024-03-31
4月の残り営業日数: 22
次の月曜日: 2024-04-01
▶ 答えを見る
EventPlanner.java
import java.time.*;
import java.time.temporal.*;

public class EventPlanner {

    static boolean isWeekend(LocalDate d) {
        return d.getDayOfWeek() == DayOfWeek.SATURDAY
            || d.getDayOfWeek() == DayOfWeek.SUNDAY;
    }

    public static void main(String[] args) {
        LocalDate base = LocalDate.of(2024, 4, 1);

        // 月末の営業日（月末が土日なら前の金曜日）
        LocalDate monthEnd = base.with(TemporalAdjusters.lastDayOfMonth());
        while (isWeekend(monthEnd)) monthEnd = monthEnd.minusDays(1);
        System.out.println("月末営業日: " + monthEnd);

        // 第1四半期末（3月末）
        LocalDate q1End = base.withMonth(3)
            .with(TemporalAdjusters.lastDayOfMonth());
        System.out.println("第1四半期末: " + q1End);

        // 残り営業日数（4/1〜4/30）
        LocalDate end = base.with(TemporalAdjusters.lastDayOfMonth());
        long bizDays = base.datesUntil(end.plusDays(1))
            .filter(d -> !isWeekend(d))
            .count();
        System.out.println("4月の残り営業日数: " + bizDays);

        // 次の月曜日（基準日以降）
        LocalDate nextMon = base.with(
            TemporalAdjusters.nextOrSame(DayOfWeek.MONDAY));
        System.out.println("次の月曜日: " + nextMon);
    }
}
解説
  • 2024-04-30 は火曜日なのでそのまま月末営業日。
  • 第1四半期末：withMonth(3).with(lastDayOfMonth()) → 2024-03-31。
  • 4月の平日数：4/1（月）〜4/30（火）で土日は6日（4/6,7,13,14,20,21,27,28）= 8日。30-8=22日。
  • 2024-04-01 は月曜日なので nextOrSame(MONDAY) = 4/1 自体。
