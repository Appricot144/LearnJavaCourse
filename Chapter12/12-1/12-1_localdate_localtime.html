Java 8 の Date/Time API とは？

Java 8 で追加された java.time パッケージの日付・時刻クラスです。
旧来の java.util.Date の問題点（可変・スレッド非安全・月が0始まりなど）を解決したイミュータブルな設計です。

主要な3クラス

📅
LocalDate
日付のみ（年・月・日）
2024-03-15
🕐
LocalTime
時刻のみ（時・分・秒）
14:30:00
📆
LocalDateTime
日付＋時刻（両方）
2024-03-15T14:30:00

インスタンスの生成

生成方法
import java.time.*;

// 現在の日時
LocalDate     today   = LocalDate.now();
LocalTime     now     = LocalTime.now();
LocalDateTime current = LocalDateTime.now();

// 指定した日時（月は1始まり！）
LocalDate     date    = LocalDate.of(2024, 3, 15);
LocalTime     time    = LocalTime.of(14, 30, 0);
LocalDateTime dt      = LocalDateTime.of(2024, 3, 15, 14, 30, 0);

// 文字列から生成
LocalDate     parsed  = LocalDate.parse("2024-03-15");
LocalTime     parsedT = LocalTime.parse("14:30:00");

System.out.println(date);    // 2024-03-15
System.out.println(time);    // 14:30
System.out.println(dt);      // 2024-03-15T14:30

フィールドの取得

各フィールドへのアクセス
LocalDate date = LocalDate.of(2024, 3, 15);

System.out.println(date.getYear());        // 2024
System.out.println(date.getMonthValue());  // 3（1始まり）
System.out.println(date.getMonth());       // MARCH（Enum）
System.out.println(date.getDayOfMonth());  // 15
System.out.println(date.getDayOfWeek());   // FRIDAY（Enum）
System.out.println(date.getDayOfYear());   // 75

LocalTime time = LocalTime.of(14, 30, 45);
System.out.println(time.getHour());        // 14
System.out.println(time.getMinute());      // 30
System.out.println(time.getSecond());      // 45

日付・時刻の計算（イミュータブル）

日付オブジェクトは変更不可（イミュータブル）です。加算・減算は新しいオブジェクトを返します。

加算・減算メソッド
LocalDate date = LocalDate.of(2024, 3, 15);

LocalDate tomorrow   = date.plusDays(1);     // 2024-03-16
LocalDate nextMonth  = date.plusMonths(1);   // 2024-04-15
LocalDate nextYear   = date.plusYears(1);    // 2025-03-15
LocalDate lastWeek   = date.minusWeeks(1);   // 2024-03-08

System.out.println(tomorrow);   // 2024-03-16
System.out.println(nextMonth);  // 2024-04-15
System.out.println(lastWeek);   // 2024-03-08

// 元のオブジェクトは変わらない！
System.out.println(date);       // 2024-03-15（変わらない）

日付の比較

比較メソッド
LocalDate d1 = LocalDate.of(2024, 1, 1);
LocalDate d2 = LocalDate.of(2024, 12, 31);

System.out.println(d1.isBefore(d2));     // true
System.out.println(d1.isAfter(d2));      // false
System.out.println(d1.isEqual(d2));      // false
System.out.println(d1.compareTo(d2));    // 負の値（d1 < d2）

// isLeapYear（うるう年判定）
System.out.println(d1.isLeapYear());     // true（2024年はうるう年）

with メソッド：特定フィールドを変更した新しいオブジェクトを作る

with メソッド
LocalDate date = LocalDate.of(2024, 3, 15);

LocalDate endOfMonth  = date.withDayOfMonth(date.lengthOfMonth()); // 3/31
LocalDate startOfYear = date.withDayOfYear(1);    // 2024-01-01
LocalDate nextYear    = date.withYear(2025);       // 2025-03-15

System.out.println(endOfMonth);   // 2024-03-31
System.out.println(startOfYear);  // 2024-01-01

LocalDate のよく使うメソッド一覧

メソッド 説明
now() 現在の日付を取得
of(y,m,d) 指定した日付を生成
plusDays/Months/Years(n) n日/月/年後を返す
minusDays/Months/Years(n) n日/月/年前を返す
isBefore/isAfter/isEqual 日付の前後比較
getDayOfWeek() 曜日（DayOfWeek Enum）
lengthOfMonth() その月の日数を返す
isLeapYear() うるう年かどうか

コード例

DateTimeExample.java
import java.time.*;
import java.time.format.DateTimeFormatter;

public class DateTimeExample {
    public static void main(String[] args) {
        LocalDate birthday  = LocalDate.of(1995, 7, 20);
        LocalDate today     = LocalDate.of(2024, 3, 15);

        System.out.println("誕生日: " + birthday);
        System.out.println("今日:   " + today);

        // 誕生日の曜日
        System.out.println("曜日: " + birthday.getDayOfWeek());

        // 次の誕生日
        LocalDate nextBirthday = birthday.withYear(today.getYear());
        if (nextBirthday.isBefore(today) || nextBirthday.isEqual(today)) {
            nextBirthday = nextBirthday.plusYears(1);
        }
        System.out.println("次の誕生日: " + nextBirthday);

        // 日付のフォーマット
        DateTimeFormatter fmt = DateTimeFormatter.ofPattern("yyyy年MM月dd日");
        System.out.println(today.format(fmt));

        // LocalDateTime
        LocalDateTime dt = LocalDateTime.of(today, LocalTime.of(9, 0));
        System.out.println("会議: " + dt);
        System.out.println("2時間後: " + dt.plusHours(2));
    }
}

解説
  • 2024-02-29 は実在する日付（2024年はうるう年）。
  • getDayOfWeek() → THURSDAY（木曜日）。
  • plusDays(1) → 2月29日の翌日は3月1日。
  • plusMonths(1) → 3月29日（2月29日の1ヶ月後）。
  • minusYears(1) → 2023年はうるう年でない → false。
  • d.isBefore(d2) → d < d2 なので true。

}

Main.java
import java.time.*;

public class Main {
    public static void main(String[] args) {
        LocalDate base = LocalDate.of(2024, 4, 1);

        System.out.println("基準日: "       + base);
        System.out.println("曜日: "         + base.getDayOfWeek());
        System.out.println("100日後: "      + base.plusDays(100));
        System.out.println("その月の最終日: " + base.withDayOfMonth(base.lengthOfMonth()));
        System.out.println("その年の最終日: " + base.withDayOfYear(base.lengthOfYear()));
        System.out.println("うるう年: "      + base.isLeapYear());
        System.out.println("翌年の同日: "    + base.plusYears(1));
        System.out.println("1週間前: "       + base.minusWeeks(1));
    }
}
解説
  • 2024-04-01 は月曜日（MONDAY）。
  • plusDays(100)：4月1日 + 100日 = 7月10日。
  • withDayOfMonth(lengthOfMonth())：4月は30日なので 2024-04-30。
  • lengthOfYear()：2024年はうるう年なので366日 → 12月31日。
  • minusWeeks(1)：4月1日の1週間前 = 3月25日。
