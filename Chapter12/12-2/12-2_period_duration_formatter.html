Period と Duration：2つの「期間」クラス

📅 Period（日付ベースの期間）
// 年・月・日 で表す
Period p = Period.of(1, 2, 3);
// → 1年2ヶ月3日

// 2つの日付の差
LocalDate d1 = LocalDate.of(2020, 1, 1);
LocalDate d2 = LocalDate.of(2024, 3, 15);
Period diff = Period.between(d1, d2);
// diff.getYears()  → 4
// diff.getMonths() → 2
// diff.getDays()   → 14
🕐 Duration（時間ベースの期間） 
// 時・分・秒・ナノ秒 で表す
Duration d = Duration.ofHours(2).plusMinutes(30);
// → 2時間30分

// 2つの時刻の差
LocalTime t1 = LocalTime.of(9, 0);
LocalTime t2 = LocalTime.of(17, 30);
Duration diff = Duration.between(t1, t2);
// diff.toHours()   → 8
// diff.toMinutes() → 510

Period の使い方

Period の操作
import java.time.*;

LocalDate birthday = LocalDate.of(1995, 7, 20);
LocalDate today    = LocalDate.of(2024, 3, 15);

Period age = Period.between(birthday, today);
System.out.println(age.getYears()  + "歳");   // 28歳
System.out.println(age.getMonths() + "ヶ月"); // 7ヶ月
System.out.println(age.getDays()   + "日");   // 23日

// 加算
LocalDate future = today.plus(Period.of(1, 3, 10));  // 1年3ヶ月10日後
System.out.println(future);  // 2025-06-25

// toTotalMonths()
long months = age.toTotalMonths();
System.out.println("経過月数: " + months);  // 343

Duration の使い方

Duration の操作
import java.time.*;

LocalTime start = LocalTime.of(9, 0);
LocalTime end   = LocalTime.of(17, 30);

Duration workTime = Duration.between(start, end);
System.out.println(workTime.toHours()   + "時間");   // 8時間
System.out.println(workTime.toMinutes() + "分");     // 510分
System.out.println(workTime.getSeconds() + "秒");    // 30600秒

// LocalDateTime での差
LocalDateTime dt1 = LocalDateTime.of(2024, 1, 1, 9, 0);
LocalDateTime dt2 = LocalDateTime.of(2024, 1, 3, 17, 30);
Duration d = Duration.between(dt1, dt2);
System.out.println("合計時間: " + d.toHours() + "時間");  // 56時間

DateTimeFormatter：日付のフォーマット

日付を任意の文字列形式に変換（format）したり、文字列を日付に変換（parse）したりします。

年y / yyyy → 2024
月M / MM → 3 / 03　  MMM → Mar　  MMMM → March
日d / dd → 5 / 05
曜日E / EEE → Wed　  EEEE → Wednesday
時（24h）H / HH → 9 / 09
分・秒mm → 05　  ss → 00
DateTimeFormatter の使い方
import java.time.*;
import java.time.format.*;

LocalDate date = LocalDate.of(2024, 3, 15);
LocalDateTime dt = LocalDateTime.of(2024, 3, 15, 14, 30, 0);

// format（日付 → 文字列）
DateTimeFormatter f1 = DateTimeFormatter.ofPattern("yyyy年MM月dd日");
DateTimeFormatter f2 = DateTimeFormatter.ofPattern("yyyy/MM/dd (E)");
DateTimeFormatter f3 = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");

System.out.println(date.format(f1));  // 2024年03月15日
System.out.println(date.format(f2));  // 2024/03/15 (Fri)
System.out.println(dt.format(f3));    // 2024-03-15 14:30:00

// parse（文字列 → 日付）
LocalDate parsed1 = LocalDate.parse("2024年03月15日", f1);
System.out.println(parsed1);  // 2024-03-15

// 定義済みフォーマッタ
System.out.println(date.format(DateTimeFormatter.ISO_LOCAL_DATE));
// 2024-03-15

よく使う定義済みフォーマッタ

定数 形式 例
ISO_LOCAL_DATE yyyy-MM-dd 2024-03-15
ISO_LOCAL_TIME HH:mm:ss 14:30:00
ISO_LOCAL_DATE_TIME yyyy-MM-dd'T'HH:mm:ss 2024-03-15T14:30:00
ofLocalizedDate(FormatStyle.FULL) ロケール依存の完全な日付 2024年3月15日金曜日

コード例

PeriodDurationExample.java
import java.time.*;
import java.time.format.*;

public class PeriodDurationExample {
    public static void main(String[] args) {
        LocalDate  start = LocalDate.of(2024, 1, 1);
        LocalDate  end   = LocalDate.of(2024, 12, 31);
        LocalTime  tStart = LocalTime.of(9, 0);
        LocalTime  tEnd   = LocalTime.of(18, 30);

        // 期間
        Period period = Period.between(start, end);
        System.out.println("期間: " + period.getMonths() + "ヶ月"
            + period.getDays() + "日");

        // 勤務時間
        Duration work = Duration.between(tStart, tEnd);
        System.out.println("勤務: " + work.toHours() + "時間"
            + (work.toMinutesPart()) + "分");

        // フォーマット
        DateTimeFormatter fmt =
            DateTimeFormatter.ofPattern("yyyy年MM月dd日（E）");
        System.out.println(start.format(fmt));
        System.out.println(end.format(fmt));

        // 文字列から変換
        LocalDate deadline = LocalDate.parse("2024/06/30",
            DateTimeFormatter.ofPattern("yyyy/MM/dd"));
        System.out.println("締切: " + deadline);
    }
}
▶ 出力結果
期間: 11ヶ月30日
勤務: 9時間30分
2024年01月01日（Mon）
2024年12月31日（Tue）
締切: 2024-06-30

        System.out.println(d.toMinutes());
        System.out.println(d.toMinutesPart());

        // フォーマット
        DateTimeFormatter fmt = DateTimeFormatter.ofPattern("dd/MM/yyyy");
        System.out.println(d2.format(fmt));
    }
}

解説
  • Period.between(2020-03-01, 2024-09-15) → 4年6ヶ月14日。toTotalMonths() = 4×12+6 = 54。
  • Duration.between(08:45, 12:00) → 3時間15分。toHours() = 3、toMinutes() = 195、toMinutesPart() = 15（時間を除いた分部分）。
  • dd/MM/yyyy で 2024-09-15 → 15/09/2024。

import java.time.*;
import java.time.format.*;

public class Q2 {
    public static void main(String[] args) {
        // 文字列から変換
        DateTimeFormatter f1 = DateTimeFormatter.ofPattern("yyyy/MM/dd");
        DateTimeFormatter f2 = DateTimeFormatter.ofPattern("MM月dd日(E)");
        DateTimeFormatter f3 = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm");

        LocalDate d = LocalDate.parse("2024/07/04", f1);
        System.out.println(d);
        System.out.println(d.format(f2));

        LocalDateTime dt = LocalDateTime.parse("2024-07-04 10:30", f3);
        System.out.println(dt);
        System.out.println(dt.plusHours(90).format(f3));

        // Period で次のイベントまでの日数
        LocalDate event = LocalDate.of(2024, 12, 25);
        Period toEvent  = Period.between(d, event);
        System.out.println("イベントまで: "
            + toEvent.getMonths() + "ヶ月" + toEvent.getDays() + "日");
    }
}

解説
  • LocalDate.parse("2024/07/04", f1) → 2024-07-04（7月4日は木曜日）。
  • dt.plusHours(90)：2024-07-04 10:30 + 90時間 = 3日18時間後 → 2024-07-08 04:30。
  • Period.between(2024-07-04, 2024-12-25) → 5ヶ月21日。
問題 12-2-3
コード記述 ⭐⭐⭐

次の条件を満たす Schedule クラスを作成してください。

  • LocalDateTime start・LocalDateTime end フィールド
  • コンストラクタ (String startStr, String endStr)："yyyy/MM/dd HH:mm" 形式の文字列から生成
  • getDuration()：所要時間（Duration）を返す
  • show()：以下の形式で出力する
▶ 期待される出力
【プロジェクト会議】
  開始: 2024年06月10日（Mon） 10:00
  終了: 2024年06月10日（Mon） 12:30
  所要: 2時間30分
【合宿研修】
  開始: 2024年08月01日（Thu） 09:00
  終了: 2024年08月03日（Sat） 18:00
  所要: 57時間0分
▶ 答えを見る
Main.java
import java.time.*;
import java.time.format.*;

public class Main {

    static class Schedule {
        String        title;
        LocalDateTime start;
        LocalDateTime end;
        static DateTimeFormatter parser =
            DateTimeFormatter.ofPattern("yyyy/MM/dd HH:mm");
        static DateTimeFormatter display =
            DateTimeFormatter.ofPattern("yyyy年MM月dd日（E） HH:mm");

        Schedule(String title, String startStr, String endStr) {
            this.title = title;
            this.start = LocalDateTime.parse(startStr, parser);
            this.end   = LocalDateTime.parse(endStr,   parser);
        }

        Duration getDuration() { return Duration.between(start, end); }

        void show() {
            Duration d = getDuration();
            System.out.println("【" + title + "】");
            System.out.println("  開始: " + start.format(display));
            System.out.println("  終了: " + end.format(display));
            System.out.println("  所要: " + d.toHours() + "時間"
                + d.toMinutesPart() + "分");
        }
    }

    public static void main(String[] args) {
        Schedule s1 = new Schedule("プロジェクト会議",
            "2024/06/10 10:00", "2024/06/10 12:30");
        Schedule s2 = new Schedule("合宿研修",
            "2024/08/01 09:00", "2024/08/03 18:00");

        s1.show();
        s2.show();
    }
}
解説
  • コンストラクタで parse を使って文字列を LocalDateTime に変換します。
  • Duration.between(start, end) で所要時間を取得します。
  • toHours() と toMinutesPart()（時間を除いた分）で表示します。
  • 合宿：2024-08-01 09:00 〜 2024-08-03 18:00 = 57時間0分。
