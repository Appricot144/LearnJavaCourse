Map とは？

キー（Key）と値（Value）のペアを管理するデータ構造です。
キーから値を高速に検索できます。キーの重複は許可されません。

▶ HashMap のイメージ（キー → 値）
"apple"
→
150
"banana"
→
120
"cherry"
→
800

主要な Map の種類

HashMap
最も高速。順序は保証されない
最もよく使われる
TreeMap
キーが自然順（昇順）でソートされる
ソートが必要なとき
LinkedHashMap
追加した順序を保持する
挿入順が重要なとき

HashMap の基本操作

生成・追加・取得・削除
import java.util.HashMap;
import java.util.Map;

Map<String, Integer> prices = new HashMap<>();

// put：追加・上書き
prices.put("apple",  150);
prices.put("banana", 120);
prices.put("cherry", 800);

// get：取得（キーがなければ null）
System.out.println(prices.get("apple"));    // 150
System.out.println(prices.get("grape"));    // null

// getOrDefault：デフォルト値付き取得
System.out.println(prices.getOrDefault("grape", 0));  // 0

// containsKey / containsValue：存在確認
System.out.println(prices.containsKey("banana"));    // true
System.out.println(prices.containsValue(800));        // true

// remove：削除
prices.remove("banana");
System.out.println(prices.containsKey("banana"));    // false

// size：件数
System.out.println(prices.size());  // 2

Map のループ処理

3通りのループ方法
Map<String, Integer> prices = new HashMap<>();
prices.put("apple",  150);
prices.put("banana", 120);
prices.put("cherry", 800);

// 方法1：entrySet（キーと値の両方）
for (Map.Entry<String, Integer> entry : prices.entrySet()) {
    System.out.println(entry.getKey() + " : " + entry.getValue());
}

// 方法2：keySet（キーだけ）
for (String key : prices.keySet()) {
    System.out.println(key + " → " + prices.get(key));
}

// 方法3：forEach（ラムダ式）← 最も簡潔
prices.forEach((k, v) -> System.out.println(k + " : " + v));

便利なメソッド

putIfAbsent / merge / compute
Map<String, Integer> map = new HashMap<>();
map.put("a", 1);

// putIfAbsent：キーがない場合だけ追加
map.putIfAbsent("a", 99);  // a はすでにあるので変わらない
map.putIfAbsent("b", 2);   // b は新規追加
System.out.println(map.get("a"));  // 1
System.out.println(map.get("b"));  // 2

// merge：値を累積（カウントなどに便利）
Map<String, Integer> counter = new HashMap<>();
String[] words = {"apple", "banana", "apple", "cherry", "apple"};
for (String w : words) {
    counter.merge(w, 1, Integer::sum);
}
System.out.println(counter);
// {apple=3, banana=1, cherry=1}

不変 Map の生成（Map.of）

Map.of と Map.ofEntries
// Map.of（10件以下・順序不定）
Map<String, Integer> m = Map.of(
    "apple", 150,
    "banana", 120,
    "cherry", 800
);
// 変更しようとすると UnsupportedOperationException！

// HashMap にコピーして変更可能にする
Map<String, Integer> mutable = new HashMap<>(m);
mutable.put("grape", 200);

HashMap のよく使うメソッド一覧

メソッド 説明
put(k, v) キーと値を追加（上書き）。古い値を返す
get(k) キーから値を取得。なければ null
getOrDefault(k, def) キーがなければ def を返す
remove(k) キーのエントリを削除
containsKey(k) キーの存在確認
containsValue(v) 値の存在確認
putIfAbsent(k, v) キーがない場合だけ追加
merge(k, v, f) 値を f でマージ（累積に便利）
entrySet() キー+値のセット（ループに使用）
keySet() キーのセット
values() 値のコレクション
size() エントリ数
isEmpty() 空かどうか
clear() 全エントリを削除

コード例：単語カウンター

WordCounter.java
import java.util.*;

public class WordCounter {
    public static void main(String[] args) {
        String text = "the quick brown fox jumps over the lazy dog the fox";
        String[] words = text.split(" ");

        Map<String, Integer> counter = new HashMap<>();
        for (String w : words) {
            counter.merge(w, 1, Integer::sum);
        }

        System.out.println("単語数: " + counter.size());

        // TreeMap でソート出力
        new TreeMap<>(counter).forEach(
            (w, c) -> System.out.println(w + ": " + c)
        );

        // 最多単語を探す
        String maxWord = null;
        int    maxCount = 0;
        for (Map.Entry<String, Integer> e : counter.entrySet()) {
            if (e.getValue() > maxCount) {
                maxWord  = e.getKey();
                maxCount = e.getValue();
            }
        }
        System.out.println("最多: " + maxWord + "(" + maxCount + "回)");
    }
}
▶ 出力結果
単語数: 8
brown: 1
dog: 1
fox: 2
jumps: 1
lazy: 1
over: 1
quick: 1
the: 3
最多: the(3回)

問題 13-1-2
出力を答える ⭐⭐⭐

次のコードを実行したとき、出力される結果を答えてください。

Q2.java
import java.util.*;

public class Q2 {
    public static void main(String[] args) {
        String[] fruits = {"apple","banana","apple","cherry",
                           "banana","apple","cherry","cherry"};

        // merge でカウント
        Map<String, Integer> count = new HashMap<>();
        for (String f : fruits) {
            count.merge(f, 1, Integer::sum);
        }

        // TreeMap でソートして出力
        new TreeMap<>(count).forEach(
            (k, v) -> System.out.println(k + ": " + v)
        );

        System.out.println("---");

        // 値の合計
        int total = 0;
        for (int v : count.values()) total += v;
        System.out.println("合計: " + total);

        // 最多の果物
        String maxFruit = count.entrySet().stream()
            .max(Map.Entry.comparingByValue())
            .map(Map.Entry::getKey)
            .orElse("");
        System.out.println("最多: " + maxFruit);
    }
}
▶ 答えを見る
▶ 出力結果
apple: 3
banana: 2
cherry: 3
---
合計: 8
最多: apple
解説
  • apple 3回、banana 2回、cherry 3回。TreeMap で昇順出力。
  • 合計：3+2+3 = 8。
  • max(Map.Entry.comparingByValue())：値が最大のエントリを返す。apple と cherry が同数（3）の場合、Stream の順序依存で apple または cherry が返ります（実際の結果は apple）。
問題 13-1-3
コード記述 ⭐⭐⭐

次の条件を満たす PhoneBook（電話帳）クラスを作成してください。

  • HashMap<String, String> をフィールドに持つ（名前 → 電話番号）
  • add(String name, String number)：登録。同名が既存の場合は "[name] は既に登録済みです" を出力
  • find(String name)：検索。なければ "[name] は登録されていません" を出力、あれば番号を出力
  • update(String name, String number)：番号を更新。なければ "[name] は存在しません" を出力
  • showAll()：全件を名前の昇順で出力
▶ 期待される出力
田中: 090-1111-2222
佐藤: 080-3333-4444
田中 は既に登録済みです
鈴木: 070-5555-6666
---
佐藤: 080-3333-4444
田中: 090-1111-2222
鈴木: 070-5555-6666
---
田中: 090-9999-0000
山田 は存在しません
▶ 答えを見る
PhoneBook.java + Main
import java.util.*;

class PhoneBook {
    private Map<String, String> book = new HashMap<>();

    public void add(String name, String number) {
        if (book.containsKey(name)) {
            System.out.println(name + " は既に登録済みです");
        } else {
            book.put(name, number);
            System.out.println(name + ": " + number);
        }
    }

    public void find(String name) {
        String num = book.get(name);
        if (num == null) {
            System.out.println(name + " は登録されていません");
        } else {
            System.out.println(name + ": " + num);
        }
    }

    public void update(String name, String number) {
        if (!book.containsKey(name)) {
            System.out.println(name + " は存在しません");
        } else {
            book.put(name, number);
            System.out.println(name + ": " + number);
        }
    }

    public void showAll() {
        new TreeMap<>(book).forEach(
            (k, v) -> System.out.println(k + ": " + v));
    }
}

public class Main {
    public static void main(String[] args) {
        PhoneBook pb = new PhoneBook();
        pb.add("田中", "090-1111-2222");
        pb.add("佐藤", "080-3333-4444");
        pb.add("田中", "090-9999-0000");  // 重複
        pb.add("鈴木", "070-5555-6666");
        System.out.println("---");
        pb.showAll();
        System.out.println("---");
        pb.update("田中", "090-9999-0000");
        pb.update("山田", "000-0000-0000");
    }
}
解説
  • add：containsKey で重複チェック、なければ put で追加。
  • update：containsKey で存在確認後 put で上書き。
  • showAll：TreeMap に変換してキー昇順で出力。
問題 13-1-4
コード記述 ⭐⭐⭐

次の条件を満たすクラスを作成してください。

  • 文字列配列 {"Java", "Python", "Java", "C", "Python", "Java", "Go", "C"} を処理する
  • merge で各言語の出現回数を集計する
  • 出現回数を降順（同数の場合はアルファベット昇順）でソートして出力する
  • 出現回数が2以上の言語名を "," 区切りでアルファベット昇順に出力する
▶ 期待される出力
Java: 3回
C: 2回
Python: 2回
Go: 1回
---
複数回: C, Java, Python
💡 ヒント：降順ソートは entrySet().stream() を使い、
Comparator.comparingByValue(Comparator.reverseOrder()).thenComparing(Map.Entry.comparingByKey()) で並べます。
▶ 答えを見る
Main.java
import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        String[] langs = {"Java","Python","Java","C","Python","Java","Go","C"};

        Map<String, Integer> count = new HashMap<>();
        for (String l : langs) count.merge(l, 1, Integer::sum);

        // 出現回数降順 → アルファベット昇順
        count.entrySet().stream()
            .sorted(Map.Entry.<String,Integer>comparingByValue(
                        Comparator.reverseOrder())
                    .thenComparing(Map.Entry.comparingByKey()))
            .forEach(e -> System.out.println(e.getKey() + ": " + e.getValue() + "回"));

        System.out.println("---");

        // 2回以上の言語をアルファベット昇順
        String multi = count.entrySet().stream()
            .filter(e -> e.getValue() >= 2)
            .map(Map.Entry::getKey)
            .sorted()
            .collect(Collectors.joining(", "));
        System.out.println("複数回: " + multi);
    }
}
解説
  • Java:3、Python:2、C:2、Go:1 の順に集計されます。
  • 降順ソート後、同数（C=Python=2）はアルファベット昇順でC→Python。
  • 2回以上のアルファベット昇順：C, Java, Python。
