HashMap / TreeMap / LinkedHashMap の比較

HashMap
⚡ 最速

ハッシュテーブルで実装。
順序の保証なし。null キー・値OK。

TreeMap
🔤 キー昇順

赤黒木で実装。キーが常に昇順。firstKey/lastKey などが使える。

LinkedHashMap
📋 挿入順保持

ハッシュ＋双方向リスト。追加した順序でイテレートされる。

TreeMap：キーがソートされる Map

TreeMap の使い方
import java.util.*;

TreeMap<String, Integer> tree = new TreeMap<>();
tree.put("banana", 120);
tree.put("apple",  150);
tree.put("cherry", 800);
tree.put("date",   200);

// 自動的にキー昇順
tree.forEach((k, v) -> System.out.println(k + ": " + v));
// apple: 150, banana: 120, cherry: 800, date: 200

// TreeMap 固有のメソッド
System.out.println(tree.firstKey());        // apple（最小キー）
System.out.println(tree.lastKey());         // date（最大キー）
System.out.println(tree.lowerKey("cherry")); // banana（cherryより小さい最大キー）
System.out.println(tree.higherKey("banana")); // cherry

// headMap：指定キー未満のサブマップ
System.out.println(tree.headMap("cherry")); // {apple=150, banana=120}

// tailMap：指定キー以上のサブマップ
System.out.println(tree.tailMap("cherry")); // {cherry=800, date=200}

LinkedHashMap：挿入順を保持する Map

LinkedHashMap の使い方
import java.util.*;

// 挿入順
Map<String, Integer> linked = new LinkedHashMap<>();
linked.put("banana", 120);
linked.put("apple",  150);
linked.put("cherry", 800);

// 挿入した順番で出力される
linked.forEach((k, v) -> System.out.println(k + ": " + v));
// banana: 120 → apple: 150 → cherry: 800

// アクセス順モード（LRUキャッシュなどに使用）
Map<String, Integer> lru = new LinkedHashMap<>(16, 0.75f, true);
lru.put("a", 1);
lru.put("b", 2);
lru.put("c", 3);
lru.get("a");   // a にアクセス → a が末尾へ
lru.forEach((k, v) -> System.out.print(k + " "));
// b c a（アクセス順）

Map と Stream API の組み合わせ

entrySet().stream() で処理
import java.util.*;
import java.util.stream.*;

Map<String, Integer> scores = Map.of(
    "Alice", 85, "Bob", 72, "Carol", 91,
    "Dave", 60, "Eve", 88
);

// 80点以上の名前を昇順で取得
List<String> passed = scores.entrySet().stream()
    .filter(e -> e.getValue() >= 80)
    .map(Map.Entry::getKey)
    .sorted()
    .collect(Collectors.toList());
System.out.println(passed);  // [Alice, Carol, Eve]

// 点数の平均
double avg = scores.values().stream()
    .mapToInt(Integer::intValue)
    .average()
    .orElse(0);
System.out.printf("平均: %.1f%n", avg);  // 79.2

// 点数降順に並べた Map（LinkedHashMap で順序保持）
Map<String, Integer> sorted = scores.entrySet().stream()
    .sorted(Map.Entry.<String,Integer>comparingByValue().reversed())
    .collect(Collectors.toMap(
        Map.Entry::getKey,
        Map.Entry::getValue,
        (a, b) -> a,
        LinkedHashMap::new
    ));
sorted.forEach((k, v) -> System.out.println(k + ": " + v));

値がリストの Map（グループ化）

Map<String, List> の操作
import java.util.*;

// 学年ごとの学生リスト
Map<Integer, List<String>> byGrade = new HashMap<>();

// computeIfAbsent でリストを自動生成
byGrade.computeIfAbsent(1, k -> new ArrayList<>()).add("Alice");
byGrade.computeIfAbsent(1, k -> new ArrayList<>()).add("Bob");
byGrade.computeIfAbsent(2, k -> new ArrayList<>()).add("Carol");
byGrade.computeIfAbsent(3, k -> new ArrayList<>()).add("Dave");
byGrade.computeIfAbsent(3, k -> new ArrayList<>()).add("Eve");

new TreeMap<>(byGrade).forEach(
    (g, list) -> System.out.println(g + "年: " + list)
);
// 1年: [Alice, Bob]
// 2年: [Carol]
// 3年: [Dave, Eve]

コード例：成績管理システム

GradeManager.java
import java.util.*;
import java.util.stream.*;

public class GradeManager {
    public static void main(String[] args) {
        Map<String, List<Integer>> grades = new LinkedHashMap<>();
        grades.put("Alice", List.of(85, 92, 78));
        grades.put("Bob",   List.of(70, 65, 80));
        grades.put("Carol", List.of(95, 88, 91));

        // 各生徒の平均点
        grades.forEach((name, scores) -> {
            double avg = scores.stream()
                .mapToInt(Integer::intValue).average().orElse(0);
            System.out.printf("%s: 平均%.1f点%n", name, avg);
        });

        System.out.println("---");

        // 平均点でランキング
        grades.entrySet().stream()
            .sorted(Comparator.comparingDouble(
                (Map.Entry<String, List<Integer>> e) ->
                    e.getValue().stream().mapToInt(Integer::intValue).average().orElse(0)
            ).reversed())
            .forEach(e -> System.out.println(e.getKey()));
    }
}

import java.util.*;

public class Q1 {
    public static void main(String[] args) {
        // TreeMap（キー昇順）
        TreeMap<Integer, String> tree = new TreeMap<>();
        tree.put(3, "three");
        tree.put(1, "one");
        tree.put(5, "five");
        tree.put(2, "two");
        tree.put(4, "four");

        System.out.println(tree.firstKey());
        System.out.println(tree.lastKey());
        System.out.println(tree.lowerKey(3));
        System.out.println(tree.higherKey(3));
        System.out.println(tree.headMap(3));
        System.out.println(tree.tailMap(3));

        System.out.println("---");

        // LinkedHashMap（挿入順）
        Map<String, Integer> linked = new LinkedHashMap<>();
        linked.put("c", 3);
        linked.put("a", 1);
        linked.put("b", 2);
        linked.forEach((k, v) -> System.out.print(k + " "));
        System.out.println();

        // 同じデータをHashMapに入れて出力（順不定なので確定しないが）
        // TreeMapで出力
        new TreeMap<>(linked).forEach((k, v) -> System.out.print(k + " "));
        System.out.println();
    }
}

解説
  • firstKey() = 1（最小）、lastKey() = 5（最大）。
  • lowerKey(3) = 2（3より小さい最大キー）、higherKey(3) = 4（3より大きい最小キー）。
  • headMap(3)：3未満のサブマップ → {1=one, 2=two}。
  • tailMap(3)：3以上のサブマップ → {3=three, 4=four, 5=five}。
  • LinkedHashMap は挿入順（c→a→b）で出力。TreeMap に変換するとキー昇順（a→b→c）。

Q2.java
import java.util.*;
import java.util.stream.*;

public class Q2 {
    public static void main(String[] args) {
        Map<String, Integer> stock = new LinkedHashMap<>();
        stock.put("りんご",  100);
        stock.put("バナナ",   50);
        stock.put("メロン",   10);
        stock.put("いちご",   30);
        stock.put("みかん",   80);

        // 在庫30以下の商品名を出力（挿入順）
        stock.entrySet().stream()
            .filter(e -> e.getValue() <= 30)
            .forEach(e -> System.out.println(
                e.getKey() + ": " + e.getValue()));

        System.out.println("---");

        // 在庫の合計と平均
        IntSummaryStatistics stats = stock.values().stream()
            .mapToInt(Integer::intValue)
            .summaryStatistics();
        System.out.println("合計: " + stats.getSum());
        System.out.println("平均: " + stats.getAverage());

        System.out.println("---");

        // computeIfAbsent で在庫0の商品を追加
        stock.computeIfAbsent("ぶどう", k -> 0);
        System.out.println(stock.get("ぶどう"));
        System.out.println(stock.size());
    }
}

解説
  • 在庫30以下：メロン(10)・いちご(30)。LinkedHashMap なので挿入順で出力。
  • 合計：100+50+10+30+80 = 270、平均：270/5 = 54.0。
  • computeIfAbsent("ぶどう", k → 0)：ぶどうは未登録なので 0 が追加。size は 6 になる。
問題 13-2-3
コード記述 ⭐⭐⭐

次の条件を満たすクラスを作成してください。

  • Map<String, List<Integer>>（科目名 → 点数リスト）を使って複数学生の成績を管理する
  • 以下の成績データを登録する（computeIfAbsent を使って List を自動生成する）
科目 点数リスト
数学 85, 72, 91, 60
英語 78, 88, 65, 95
国語 90, 70, 80, 75
▶ 期待される出力
国語: 平均78.8点 最高90点 最低70点
数学: 平均77.0点 最高91点 最低60点
英語: 平均81.5点 最高95点 最低65点
---
全科目で80点以上の科目数: 1
最高平均科目: 英語
💡 ヒント：科目の統計は summaryStatistics() で取得します。
TreeMap で科目名昇順に出力します。
▶ 答えを見る
Main.java
import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        Map<String, List<Integer>> grades = new HashMap<>();

        int[][] data = {{85,72,91,60},{78,88,65,95},{90,70,80,75}};
        String[] subjects = {"数学", "英語", "国語"};

        for (int i = 0; i < subjects.length; i++) {
            List<Integer> list = grades.computeIfAbsent(
                subjects[i], k -> new ArrayList<>());
            for (int score : data[i]) list.add(score);
        }

        // 科目名昇順で統計表示
        new TreeMap<>(grades).forEach((subj, scores) -> {
            IntSummaryStatistics s = scores.stream()
                .mapToInt(Integer::intValue).summaryStatistics();
            System.out.printf("%s: 平均%.1f点 最高%d点 最低%d点%n",
                subj, s.getAverage(), s.getMax(), s.getMin());
        });

        System.out.println("---");

        // 平均80点以上の科目数
        long highCount = grades.values().stream()
            .filter(scores -> scores.stream()
                .mapToInt(Integer::intValue).average().orElse(0) >= 80)
            .count();
        System.out.println("全科目で80点以上の科目数: " + highCount);

        // 最高平均科目
        String best = grades.entrySet().stream()
            .max(Comparator.comparingDouble(e ->
                e.getValue().stream().mapToInt(Integer::intValue).average().orElse(0)))
            .map(Map.Entry::getKey).orElse("");
        System.out.println("最高平均科目: " + best);
    }
}
解説
  • 国語：(90+70+80+75)/4 = 78.75 → 78.8、最高90、最低70。
  • 数学：(85+72+91+60)/4 = 77.0。英語：(78+88+65+95)/4 = 81.5。
  • 平均80点以上の科目：英語(81.5)のみ = 1科目。
  • 最高平均は英語(81.5)。
