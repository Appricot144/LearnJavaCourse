📖 解説

java.util.function パッケージとは？

Java 8 で追加された標準の関数型インターフェース群です。
自分で定義しなくても、よく使うパターンがすでに用意されています。

Function<T, R>
R apply(T t)
T を受け取り R を返す。変換処理に使う
Function<String, Integer> len
    = s -> s.length();
len.apply("hello"); // 5
Predicate<T>
boolean test(T t)
T を受け取り boolean を返す。条件判定に使う
Predicate<Integer> isEven
    = n -> n % 2 == 0;
isEven.test(4); // true
Consumer<T>
void accept(T t)
T を受け取り何も返さない。出力・副作用処理に使う
Consumer<String> print
    = s -> System.out.println(s);
print.accept("Hi"); // Hi
Supplier<T>
T get()
引数なしで T を返す。値の生成・遅延初期化に使う
Supplier<String> hello
    = () -> "Hello!";
hello.get(); // "Hello!"

よく使うバリエーション

インターフェースシグネチャ用途
Function<T,R>R apply(T t)T → R の変換
BiFunction<T,U,R>R apply(T t, U u)T,U → R の変換
UnaryOperator<T>T apply(T t)T → T（同じ型）
BinaryOperator<T>T apply(T t1, T t2)T,T → T
Predicate<T>boolean test(T t)条件判定
BiPredicate<T,U>boolean test(T t, U u)2引数の条件判定
Consumer<T>void accept(T t)副作用処理
Supplier<T>T get()値の生成

Predicate の合成メソッド

and / or / negate で条件を組み合わせる
import java.util.function.Predicate;

Predicate<Integer> isPositive = n -> n > 0;
Predicate<Integer> isEven    = n -> n % 2 == 0;

// and：両方満たす
Predicate<Integer> isPositiveEven = isPositive.and(isEven);
System.out.println(isPositiveEven.test(4));   // true
System.out.println(isPositiveEven.test(-2));  // false

// or：どちらか満たす
Predicate<Integer> eitherOr = isPositive.or(isEven);
System.out.println(eitherOr.test(-2));  // true（偶数なので）

// negate：反転
Predicate<Integer> notPositive = isPositive.negate();
System.out.println(notPositive.test(-5));  // true

Function の合成メソッド

andThen / compose でパイプライン処理
import java.util.function.Function;

Function<String, String> trim    = String::trim;
Function<String, String> upper   = String::toUpperCase;
Function<String, Integer> length = String::length;

// andThen：左 → 右の順に適用
Function<String, String> trimThenUpper = trim.andThen(upper);
System.out.println(trimThenUpper.apply("  hello  ")); // HELLO

// 連鎖
Function<String, Integer> pipeline = trim.andThen(upper).andThen(length);
System.out.println(pipeline.apply("  hello  ")); // 5

コード例：FI を使った汎用処理

FunctionExample.java
import java.util.function.*;

public class FunctionExample {
    public static void main(String[] args) {
        // Function
        Function<String, Integer> toLength = String::length;
        Function<Integer, String> stars    = n -> "*".repeat(n);

        System.out.println(toLength.apply("Java"));  // 4
        System.out.println(stars.apply(5));           // *****
        System.out.println(toLength.andThen(stars).apply("Hello")); // *****

        // Predicate
        Predicate<String> longWord = s -> s.length() > 4;
        Predicate<String> startsA  = s -> s.startsWith("A");
        System.out.println(longWord.and(startsA).test("Apple")); // true
        System.out.println(longWord.and(startsA).test("Art"));   // false

        // Consumer
        Consumer<String> logger = s -> System.out.println("[LOG] " + s);
        logger.accept("処理開始");  // [LOG] 処理開始

        // Supplier
        Supplier<Double> random = Math::random;
        System.out.println(random.get() >= 0);  // true（必ず0以上）
    }
}

import java.util.function.*;

public class Q1 {
    public static void main(String[] args) {
        Function<Integer, Integer> doubleIt = n -> n * 2;
        Function<Integer, Integer> addTen   = n -> n + 10;
        Function<Integer, String>  toStr    = n -> "value=" + n;

        System.out.println(doubleIt.apply(5));
        System.out.println(addTen.apply(5));
        System.out.println(doubleIt.andThen(addTen).apply(5));
        System.out.println(addTen.andThen(doubleIt).apply(5));
        System.out.println(doubleIt.andThen(toStr).apply(7));

        Predicate<String> isEmpty  = String::isEmpty;
        Predicate<String> isLong   = s -> s.length() > 5;

        System.out.println(isEmpty.test(""));
        System.out.println(isEmpty.negate().test("hi"));
        System.out.println(isLong.test("Hello"));
        System.out.println(isEmpty.or(isLong).test("HelloWorld"));
    }
}

解説
  • doubleIt.apply(5) = 10。
  • doubleIt.andThen(addTen).apply(5) = 5×2=10 → 10+10=20。
  • addTen.andThen(doubleIt).apply(5) = 5+10=15 → 15×2=30。

import java.util.function.*;

public class Q2 {

    static <T> void forEach(T[] arr, Consumer<T> action) {
        for (T item : arr) action.accept(item);
    }

    static <T> T[] filterFirst(T[] arr, Predicate<T> pred, T[] result) {
        int idx = 0;
        for (T item : arr) {
            if (pred.test(item) && idx < result.length) result[idx++] = item;
        }
        return result;
    }

    public static void main(String[] args) {
        String[] words = {"apple", "ant", "banana", "avocado", "cherry"};

        // Consumer で出力
        Consumer<String> print  = s -> System.out.print(s + " ");
        Consumer<String> upper  = s -> System.out.print(s.toUpperCase() + " ");

        forEach(words, print);
        System.out.println();

        // Predicate で絞り込み
        Predicate<String> startsA = s -> s.startsWith("a");
        Predicate<String> longWord = s -> s.length() > 4;

        forEach(words, upper.andThen(s -> {}));  // 何もしない合成
        System.out.println();

        // andThen で Consumer を合成
        Consumer<String> printAndCount =
            print.andThen(s -> System.out.print("(" + s.length() + ") "));
        printAndCount.accept("Java");
        System.out.println();

        System.out.println(startsA.and(longWord).test("apple"));
        System.out.println(startsA.and(longWord).test("ant"));
        System.out.println(startsA.and(longWord).test("avocado"));
    }
}

解説
  • 1行目：forEach(words, print) で全単語をスペース区切りで出力。
  • 2行目：upper.andThen(s -> {}) は upper（大文字出力）の後に何もしない Consumer を合成。upper が実行されるが forEach の2回目では upper だけが動く → 大文字出力後改行。
  • 3行目：print.andThen(...) で "Java" と文字数 (4) を出力。
  • "apple"：a始まり かつ 長さ5 → true。"ant"：a始まり だが 長さ3 → false。"avocado"：a始まり かつ 長さ7 → true。
問題 10-2-3
コード記述 ⭐⭐⭐

次の条件を満たすクラスを作成してください。

  • Function<String, Integer> toScore：文字列の文字コードの合計を返す（各文字の (int) キャスト値の合計）
  • Predicate<Integer> isPassing：300以上なら true
  • Consumer<String> logger："[SCORE] 文字列: X点 → 合否" の形式で出力
  • Supplier<String> defaultName："Anonymous" を返す
▶ 期待される出力
[SCORE] Java: 379点 → 合格
[SCORE] Hi: 177点 → 不合格
デフォルト名: Anonymous
💡 ヒント："Java" の文字コード合計：J(74)+a(97)+v(118)+a(97) = 386…
実際の出力値はコードを実行して確認してください。logger 内で toScore と isPassing を使います。
▶ 答えを見る
Main.java
import java.util.function.*;

public class Main {
    public static void main(String[] args) {
        Function<String, Integer> toScore = s -> {
            int sum = 0;
            for (char c : s.toCharArray()) sum += (int) c;
            return sum;
        };

        Predicate<Integer> isPassing = score -> score >= 300;

        Consumer<String> logger = s -> {
            int score = toScore.apply(s);
            String result = isPassing.test(score) ? "合格" : "不合格";
            System.out.println("[SCORE] " + s + ": " + score + "点 → " + result);
        };

        Supplier<String> defaultName = () -> "Anonymous";

        logger.accept("Java");
        logger.accept("Hi");
        System.out.println("デフォルト名: " + defaultName.get());
    }
}
解説
  • toScore：Function で文字コードの合計を計算します。
  • isPassing：Predicate でスコアが300以上かを判定します。
  • logger：Consumer の中で toScore と isPassing を組み合わせます。
  • defaultName：Supplier で固定の文字列を供給します。
問題 10-2-4
コード記述 ⭐⭐⭐

次の条件を満たすクラスを作成してください。

  • BinaryOperator<Integer> を使って加算・乗算・最大値の3つを定義する
  • UnaryOperator<String> を使って「先頭大文字化」と「末尾に "!" 付加」を定義する
  • 静的メソッド reduce(int[] arr, int init, BinaryOperator<Integer> op) を定義する。init から始めて配列の全要素に op を適用した結果を返す
▶ 期待される出力
合計: 15
積: 120
最大: 5
先頭大文字: Hello
感嘆符付き: Hello!
▶ 答えを見る
Main.java
import java.util.function.*;

public class Main {

    static int reduce(int[] arr, int init, BinaryOperator<Integer> op) {
        int result = init;
        for (int n : arr) result = op.apply(result, n);
        return result;
    }

    public static void main(String[] args) {
        BinaryOperator<Integer> add = (a, b) -> a + b;
        BinaryOperator<Integer> mul = (a, b) -> a * b;
        BinaryOperator<Integer> max = (a, b) -> a > b ? a : b;

        int[] nums = {1, 2, 3, 4, 5};

        System.out.println("合計: " + reduce(nums, 0,   add));
        System.out.println("積: "   + reduce(nums, 1,   mul));
        System.out.println("最大: " + reduce(nums, nums[0], max));

        UnaryOperator<String> capitalize =
            s -> s.substring(0,1).toUpperCase() + s.substring(1).toLowerCase();
        UnaryOperator<String> exclaim = s -> s + "!";

        String word = "hello";
        String result = capitalize.andThen(exclaim).apply(word);
        System.out.println("先頭大文字: " + capitalize.apply(word));
        System.out.println("感嘆符付き: " + result);
    }
}
解説
  • reduce(nums, 0, add)：0+1+2+3+4+5 = 15。
  • reduce(nums, 1, mul)：1×1×2×3×4×5 = 120。
  • reduce(nums, nums[0], max)：最大値 = 5。
  • UnaryOperator は入出力が同じ型の Function です。andThen で合成できます。
