メソッド参照とは？

ラムダ式が既存のメソッドをそのまま呼び出すだけのとき、:: 演算子で
より簡潔に書ける記法です。

ラムダ式 → メソッド参照への書き換え
// ラムダ式
Function<String, Integer> len1 = s -> s.length();

// メソッド参照（同じ意味）
Function<String, Integer> len2 = String::length;

メソッド参照の4種類

1
静的メソッド参照
クラス名::静的メソッド名
例：Integer::parseInt、Math::abs、System.out::println
Function<String, Integer> parse = Integer::parseInt;
parse.apply("42");  // 42
2
特定インスタンスのメソッド参照
インスタンス変数::メソッド名
例：System.out::println、str::startsWith
Consumer<String> print = System.out::println;
print.accept("Hello");  // Hello
3
任意インスタンスのメソッド参照
クラス名::インスタンスメソッド名
例：String::toLowerCase、String::length（引数のインスタンスを対象にする）
Function<String, String> lower = String::toLowerCase;
lower.apply("HELLO");  // hello
4
コンストラクタ参照
クラス名::new
例：ArrayList::new、StringBuilder::new
Supplier<StringBuilder> sb = StringBuilder::new;
sb.get();  // 新しい StringBuilder

ラムダ式 vs メソッド参照の対応表

ラムダ式メソッド参照種類
s -> Integer.parseInt(s)Integer::parseInt静的
s -> System.out.println(s)System.out::println特定インスタンス
s -> s.toLowerCase()String::toLowerCase任意インスタンス
(s, n) -> s.substring(n)String::substring任意インスタンス（2引数）
() -> new ArrayList<>()ArrayList::newコンストラクタ
⚠️ メソッド参照が使える条件
ラムダ式の処理が「引数をそのままメソッドに渡すだけ」のとき使えます。
加工や条件分岐がある場合はラムダ式のままが読みやすいです。

Arrays.sort とメソッド参照

ソートにメソッド参照を活用
import java.util.Arrays;

String[] words = {"banana", "apple", "cherry", "date"};

// ラムダ式
Arrays.sort(words, (a, b) -> a.compareTo(b));

// メソッド参照（同じ意味）
Arrays.sort(words, String::compareTo);

System.out.println(Arrays.toString(words));
// → [apple, banana, cherry, date]

// 逆順（Comparator.reverseOrder()）
Arrays.sort(words, Comparator.reverseOrder());
System.out.println(Arrays.toString(words));
// → [date, cherry, banana, apple]

コード例：様々なメソッド参照の活用

MethodRefExample.java
import java.util.function.*;
import java.util.Arrays;

public class MethodRefExample {
    static String shout(String s) {
        return s.toUpperCase() + "!!!";
    }

    public static void main(String[] args) {
        // 静的メソッド参照
        Function<String, String> shoutRef = MethodRefExample::shout;
        System.out.println(shoutRef.apply("hello"));  // HELLO!!!

        // 任意インスタンスのメソッド参照
        Function<String, Integer> lenRef    = String::length;
        Function<String, String>  upperRef  = String::toUpperCase;
        Function<String, Boolean> emptyRef  = String::isEmpty;

        System.out.println(lenRef.apply("Java"));    // 4
        System.out.println(upperRef.apply("java"));  // JAVA
        System.out.println(emptyRef.apply(""));      // true

        // 特定インスタンスのメソッド参照
        String prefix = "Hello, ";
        Function<String, String> greet = prefix::concat;
        System.out.println(greet.apply("World"));  // Hello, World

        // コンストラクタ参照
        Function<String, StringBuilder> sbFactory = StringBuilder::new;
        StringBuilder sb = sbFactory.apply("initial");
        System.out.println(sb.toString());  // initial

        // Consumer でメソッド参照
        String[] fruits = {"apple", "banana", "cherry"};
        Arrays.sort(fruits, String::compareTo);
        Arrays.stream(fruits).forEach(System.out::println);
    }
}

Q1.java
import java.util.function.*;
import java.util.Arrays;

public class Q1 {

    static int doubleIt(int n) { return n * 2; }

    public static void main(String[] args) {
        // 静的メソッド参照
        Function<String, Integer>  parseInt  = Integer::parseInt;
        Function<Double, Double>   absVal    = Math::abs;
        Function<Integer, Integer> doubleRef = Q1::doubleIt;

        System.out.println(parseInt.apply("123"));
        System.out.println(absVal.apply(-3.14));
        System.out.println(doubleRef.apply(7));

        // 任意インスタンスのメソッド参照
        Function<String, String>  trim  = String::trim;
        Function<String, Integer> len   = String::length;
        Predicate<String>         blank = String::isBlank;

        System.out.println(trim.apply("  hello  "));
        System.out.println(len.apply("Java"));
        System.out.println(blank.test("   "));
        System.out.println(blank.test("hi"));

        // Consumer でメソッド参照
        String[] words = {"cat", "apple", "bee"};
        Arrays.sort(words, String::compareTo);
        Arrays.stream(words).forEach(System.out::println);
    }
}

▶ 出力結果

hello

true
false
apple

cat
解説
  • Integer::parseInt で "123" → 123。
  • Math::abs で -3.14 → 3.14。
  • Q1::doubleIt で 7 × 2 = 14。
  • String::trim で前後空白除去 → "hello"。
  • String::isBlank：" " は空白のみなので true、"hi" は false。
  • String::compareTo でアルファベット昇順ソート → apple, bee, cat。
問題 10-3-2
出力を答える ⭐⭐⭐

次のコードを実行したとき、出力される結果を答えてください。

Q2.java
import java.util.function.*;
import java.util.Arrays;

public class Q2 {

    static class Person {
        String name; int age;
        Person(String name, int age) { this.name = name; this.age = age; }
        String getName() { return name; }
        int    getAge()  { return age; }
        String greet()   { return "Hi, I'm " + name + "!"; }

        @Override public String toString() { return name + "(" + age + ")"; }
    }

    public static void main(String[] args) {
        BiFunction<String, Integer, Person> maker = Person::new;

        Person p1 = maker.apply("Alice", 30);
        Person p2 = maker.apply("Bob",   25);
        Person p3 = maker.apply("Carol", 28);

        Person[] people = { p1, p2, p3 };

        // 年齢順ソート（メソッド参照でなくラムダ）
        Arrays.sort(people, (a, b) -> a.getAge() - b.getAge());
        System.out.println(Arrays.toString(people));

        // 名前でソート（メソッド参照）
        Arrays.sort(people, Comparator.comparing(Person::getName));
        System.out.println(Arrays.toString(people));

        // Consumer でメソッド参照
        Consumer<Person> greetAll = p -> System.out.println(p.greet());
        Arrays.stream(people).forEach(greetAll);
    }
}
▶ 答えを見る
▶ 出力結果
[Bob(25), Carol(28), Alice(30)]
[Alice(30), Bob(25), Carol(28)]
Hi, I'm Alice!
Hi, I'm Bob!
Hi, I'm Carol!
解説
  • Person::new はコンストラクタ参照（BiFunction<String,Integer,Person>）。
  • 年齢順：Bob(25) → Carol(28) → Alice(30)。
  • Comparator.comparing(Person::getName) は名前でソート → Alice, Bob, Carol のアルファベット順。
  • greetAll Consumer でアルファベット順（Alice, Bob, Carol）に挨拶を出力。
問題 10-3-3
コード記述 ⭐⭐⭐

次の条件を満たすクラスを作成してください。

  • Product クラス：name（String）・price（int）フィールド。コンストラクタ・getter・toString() を定義する
  • BiFunction<String, Integer, Product> を使い、コンストラクタ参照でインスタンスを生成する
  • 3つの商品を生成し、価格の安い順に並び替えて出力する（Comparator.comparingInt と Product::getPrice を使う）
  • 全商品を System.out::println で出力する
  • 最高価格の商品名を Function<Product, String> と Product::getName で取得して出力する
▶ 期待される出力
バナナ(120円)
りんご(200円)
メロン(800円)
最高価格商品: メロン
▶ 答えを見る
Main.java
import java.util.function.*;
import java.util.Arrays;

public class Main {

    static class Product {
        String name; int price;
        Product(String name, int price) { this.name=name; this.price=price; }
        String getName()  { return name; }
        int    getPrice() { return price; }
        @Override public String toString() { return name+"("+price+"円)"; }
    }

    public static void main(String[] args) {
        BiFunction<String, Integer, Product> maker = Product::new;

        Product[] products = {
            maker.apply("りんご", 200),
            maker.apply("メロン",  800),
            maker.apply("バナナ",  120)
        };

        // 価格の安い順にソート
        Arrays.sort(products, Comparator.comparingInt(Product::getPrice));

        // 出力
        Arrays.stream(products).forEach(System.out::println);

        // 最高価格（ソート後の最後）
        Function<Product, String> getName = Product::getName;
        System.out.println("最高価格商品: "
            + getName.apply(products[products.length - 1]));
    }
}
解説
  • Product::new（コンストラクタ参照）で BiFunction 経由でインスタンスを生成します。
  • Comparator.comparingInt(Product::getPrice) で価格昇順ソートします。
  • System.out::println（特定インスタンスのメソッド参照）で出力します。
  • Product::getName（任意インスタンスのメソッド参照）で名前を取得します。
