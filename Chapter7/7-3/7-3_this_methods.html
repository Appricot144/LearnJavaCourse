this の3つの使い方

this はクラスのインスタンス自身を指すキーワードです。主に3つの場面で使われます。

1
フィールドと引数の区別
this.name = name;
引数名とフィールド名が同じとき、this.フィールド名 でフィールドを指定する
2
自分自身を返す（メソッドチェーン）
return this;
メソッドの戻り値として this を返すと、メソッドを連続して呼び出せる
3
別コンストラクタへの委譲
this(side, side);
コンストラクタの先頭行で別のコンストラクタを呼び出し、処理を再利用する

使い方1：フィールドと引数の区別

this で区別する
public class Person {
    String name;
    int    age;

    public Person(String name, int age) {
        this.name = name;  // this.name → フィールド
        this.age  = age;   //      name → 引数
    }

    public void setName(String name) {
        this.name = name;  // セッターでも同様
    }
}

使い方2：メソッドチェーン（this を返す）

メソッドが this を返すことで、メソッドを連続して呼び出せます。

▶ return this でメソッドを連鎖できる
new Builder()

.

メソッドチェーンの実装
public class Builder {
    String text = "";

    public Builder add(String s) {
        text += s;
        return this;  // 自分自身を返す
    }

    public Builder addLine(String s) {
        text += s + "\n";
        return this;
    }

    public String build() { return text; }
}

String result = new Builder()
    .add("Hello")
    .add(", ")
    .addLine("World!")
    .build();

System.out.println(result);  // Hello, World!

使い方3：コンストラクタの委譲（this()）

Rectangle のコンストラクタ委譲
public class Rectangle {
    double width;
    double height;

    // メインコンストラクタ
    public Rectangle(double width, double height) {
        this.width  = width;
        this.height = height;
    }

    // 正方形用（委譲）
    public Rectangle(double side) {
        this(side, side);  // ← メインコンストラクタへ委譲
    }

    // デフォルト
    public Rectangle() {
        this(1.0);          // ← 上の委譲コンストラクタへ委譲
    }
}

Rectangle r1 = new Rectangle(4.0, 6.0);  // 4×6
Rectangle r2 = new Rectangle(5.0);       // 5×5
Rectangle r3 = new Rectangle();          // 1×1
⚠️ this() はコンストラクタの先頭行に書く
this(引数) はコンストラクタの最初の文でなければなりません。
他の処理の後に書くとコンパイルエラーになります。

アクセサメソッド（getter / setter）

フィールドを private にして、getter / setter で制御するのがカプセル化の基本です。

Circle クラスの getter / setter
public class Circle {
    private double radius;  // private で外から直接アクセス不可

    public Circle(double radius) {
        setRadius(radius);  // setter 経由で初期化（検証を通す）
    }

    public double getRadius() { return radius; }  // getter

    public void setRadius(double radius) {        // setter
        if (radius < 0) {
            throw new IllegalArgumentException("半径は0以上にしてください");
        }
        this.radius = radius;
    }

    public double getArea() {
        return Math.PI * radius * radius;
    }
}

インスタンスメソッド vs クラスメソッド（static）

種類 宣言 呼び出し フィールドアクセス
インスタンスメソッド public void method() obj.method() フィールドにアクセス可
クラスメソッド（static） public static void method() ClassName.method() static フィールドのみ

コード例：Pizza ビルダー

Pizza.java
public class Pizza {
    String  size;
    String  crust;
    boolean cheese;
    boolean pepperoni;

    public Pizza(String size) {
        this.size      = size;
        this.crust     = "thin";
        this.cheese    = true;
        this.pepperoni = false;
    }

    public Pizza crust(String crust) {
        this.crust = crust;
        return this;
    }

    public Pizza pepperoni() {
        this.pepperoni = true;
        return this;
    }

    public void order() {
        System.out.println(size + "サイズ / " + crust
            + "クラスト / " + (cheese ? "チーズあり" : "チーズなし")
            + " / " + (pepperoni ? "ペパロニあり" : "ペパロニなし"));
    }
}

// Main
new Pizza("M").crust("thick").pepperoni().order();
// → Mサイズ / thickクラスト / チーズあり / ペパロニあり

問題 7-3-1
出力を答える ⭐⭐☆

次のコードを実行したとき、出力される結果を答えてください。

Q1.java
class Vector2D {
    double x;
    double y;

    public Vector2D(double x, double y) {
        this.x = x;
        this.y = y;
    }

    public Vector2D add(Vector2D other) {
        return new Vector2D(this.x + other.x, this.y + other.y);
    }

    public Vector2D scale(double factor) {
        return new Vector2D(this.x * factor, this.y * factor);
    }

    public void show() {
        System.out.println("(" + x + ", " + y + ")");
    }
}

public class Q1 {
    public static void main(String[] args) {
        Vector2D v1 = new Vector2D(1.0, 2.0);
        Vector2D v2 = new Vector2D(3.0, 4.0);

        v1.show();
        v2.show();

        Vector2D v3 = v1.add(v2);
        v3.show();

        Vector2D v4 = v1.scale(3.0);
        v4.show();

        v1.show();  // v1 は変化しない
    }
}
▶ 答えを見る
▶ 出力結果
(1.0, 2.0)
(3.0, 4.0)
(4.0, 6.0)
(3.0, 6.0)
(1.0, 2.0)
解説
  • v1.add(v2) は v1 と v2 を足した新しいインスタンスを返します。v1 自体は変化しません。
  • v1.scale(3.0) も同様に新しいインスタンスを返します（1.0×3=3.0、2.0×3=6.0）。
  • 最後の v1.show() は元のまま (1.0, 2.0)。
問題 7-3-2
出力を答える ⭐⭐☆

次のコードを実行したとき、出力される結果を答えてください。

Q2.java
class Config {
    String host;
    int    port;
    int    timeout;

    public Config() {
        this.host    = "localhost";
        this.port    = 8080;
        this.timeout = 30;
    }

    public Config host(String host) {
        this.host = host;
        return this;
    }

    public Config port(int port) {
        this.port = port;
        return this;
    }

    public Config timeout(int timeout) {
        this.timeout = timeout;
        return this;
    }

    public void show() {
        System.out.println(host + ":" + port
            + " (timeout=" + timeout + "s)");
    }
}

public class Q2 {
    public static void main(String[] args) {
        Config c1 = new Config();
        c1.show();

        Config c2 = new Config()
            .host("example.com")
            .port(443)
            .timeout(60);
        c2.show();

        Config c3 = new Config().port(9000);
        c3.show();
    }
}
▶ 答えを見る
▶ 出力結果
localhost:8080 (timeout=30s)
example.com:443 (timeout=60s)
localhost:9000 (timeout=30s)
解説
  • c1 はデフォルト値のまま（localhost:8080 timeout=30）。
  • c2 は host・port・timeout をすべてメソッドチェーンで設定。各メソッドは this を返すため連続呼び出しが可能。
  • c3 は port だけ 9000 に変更。host と timeout はデフォルト値のまま。
問題 7-3-3
コード記述 ⭐⭐⭐

ShoppingCart（買い物かご）クラスを作成してください。

要素 内容
フィールド String[] items、int[] prices、int count
コンストラクタ 引数なし。配列を最大10個で初期化、count=0
addItem(String, int) 商品を追加。満杯なら "かごがいっぱいです" を出力。追加後メッセージを出力。this を返す
total() 合計金額を int で返す
showCart() 全商品と合計を出力
▶ 期待される出力
[りんご] を追加（150円）
[パン] を追加（200円）
[牛乳] を追加（180円）
---
りんご: 150円
パン: 200円
牛乳: 180円
合計: 530円
💡 ヒント：addItem が this を返すことで、
cart.addItem("りんご", 150).addItem("パン", 200).addItem("牛乳", 180); のようにチェーンできます。
▶ 答えを見る
ShoppingCart.java + Main.java
class ShoppingCart {
    String[] items;
    int[]    prices;
    int      count;

    public ShoppingCart() {
        items  = new String[10];
        prices = new int[10];
        count  = 0;
    }

    public ShoppingCart addItem(String name, int price) {
        if (count >= 10) {
            System.out.println("かごがいっぱいです");
            return this;
        }
        items[count]  = name;
        prices[count] = price;
        count++;
        System.out.println("[" + name + "] を追加（" + price + "円）");
        return this;
    }

    public int total() {
        int sum = 0;
        for (int i = 0; i < count; i++) sum += prices[i];
        return sum;
    }

    public void showCart() {
        for (int i = 0; i < count; i++) {
            System.out.println(items[i] + ": " + prices[i] + "円");
        }
        System.out.println("合計: " + total() + "円");
    }
}

public class Main {
    public static void main(String[] args) {
        ShoppingCart cart = new ShoppingCart();

        cart.addItem("りんご", 150)
            .addItem("パン",   200)
            .addItem("牛乳",   180);

        System.out.println("---");
        cart.showCart();
    }
}
解説
  • addItem は this を返すため、メソッドチェーンで連続呼び出しできます。
  • total() は prices 配列を count の個数だけ合計します。
  • showCart() は items[i] と prices[i] を出力し、最後に total() で合計を表示します。
