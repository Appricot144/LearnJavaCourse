コンストラクタとは？

インスタンス生成時に自動的に呼ばれる特別なメソッドです。
主にフィールドの初期値を設定するために使います。

▶ コンストラクタの構造（戻り値の型がない点が通常メソッドと異なる）
public
Dog
(String name, int age, String breed)
{ ... }
アクセス
修飾子 クラス名と
同じ名前 引数（フィールドの初期値） 処理内容

コンストラクタなし vs あり

❌ コンストラクタなし（毎回設定が必要） 
Dog pochi = new Dog();
pochi.name  = "ポチ";
pochi.age   = 3;
pochi.breed = "柴犬";
✅ コンストラクタあり（1行で完結） 
Dog pochi = new Dog(
    "ポチ", 3, "柴犬"
);

コンストラクタの定義と this キーワード

Dog クラスのコンストラクタ
public class Dog {
    String name;
    int    age;
    String breed;

    public Dog(String name, int age, String breed) {
        this.name  = name;   // this.name → フィールド
        this.age   = age;    //      name → 引数
        this.breed = breed;
    }

    public void bark() {
        System.out.println(name + ": ワン！");
    }
}

// 呼び出し
Dog pochi = new Dog("ポチ", 3, "柴犬");
pochi.bark();  // ポチ: ワン！
⚠️ this を忘れると？
引数名とフィールド名が同じ場合、this をつけないと引数が自分自身に代入されるだけで
フィールドに値が入りません。必ず this.フィールド名 = 引数名; と書きましょう。

デフォルトコンストラクタ

コンストラクタ定義の有無による違い
// コンストラクタを定義しない場合 → Java が自動生成
Dog dog = new Dog();     // OK（引数なしで生成できる）

// 引数ありコンストラクタだけを定義した場合
public Dog(String name) { ... }

Dog dog = new Dog();          // ❌ コンパイルエラー！
Dog dog = new Dog("ポチ");   // ✅ OK
💡 引数ありと引数なしの両方を使いたい場合
コンストラクタをオーバーロードして両方定義します。
public Dog() { ... } と public Dog(String name) { ... } を両方書けばOKです。

コンストラクタのオーバーロード

複数のコンストラクタ＋this() による委譲
public class Person {
    String name;
    int    age;
    String email;

    // コンストラクタ1：全フィールド
    public Person(String name, int age, String email) {
        this.name  = name;
        this.age   = age;
        this.email = email;
    }

    // コンストラクタ2：名前と年齢（email は空文字）
    public Person(String name, int age) {
        this(name, age, "");   // コンストラクタ1に委譲
    }

    // コンストラクタ3：名前のみ
    public Person(String name) {
        this(name, 0, "");     // コンストラクタ1に委譲
    }
}

static フィールド（クラス変数）

static を付けたフィールドはクラス全体で共有されます。

▶ static フィールドは全インスタンスで共有される
Counter クラス
static total = 3
→
c1

static フィールドの例
public class Counter {
    static int total = 0;  // クラス変数（共有）
    int id;                // インスタンス変数（各自が持つ）

    public Counter() {
        total++;
        id = total;
    }
}

Counter c1 = new Counter();
Counter c2 = new Counter();
Counter c3 = new Counter();

System.out.println(Counter.total);  // 3（クラス名でアクセス）
System.out.println(c1.id);          // 1
System.out.println(c3.id);          // 3

コード例：Student クラス

Student.java + Main.java
public class Student {
    String name;
    int    grade;
    double gpa;

    public Student(String name, int grade, double gpa) {
        this.name  = name;
        this.grade = grade;
        this.gpa   = gpa;
    }

    public Student(String name) {
        this(name, 1, 0.0);  // 上のコンストラクタに委譲
    }

    public void showProfile() {
        System.out.println(name + "（" + grade + "年・GPA:" + gpa + "）");
    }
}

// Main
Student s1 = new Student("田中", 3, 3.8);
Student s2 = new Student("山田");   // grade=1, gpa=0.0

s1.showProfile();  // 田中（3年・GPA:3.8）
s2.showProfile();  // 山田（1年・GPA:0.0）
▶ 出力結果
田中（3年・GPA:3.8）
山田（1年・GPA:0.0）

class Book {
    String title;
    String author;
    int    price;

    public Book(String title, String author, int price) {
        this.title  = title;
        this.author = author;
        this.price  = price;
    }

    public void show() {
        System.out.println("「" + title + "」"
            + author + " (" + price + "円)");
    }
}

public class Q1 {
    public static void main(String[] args) {
        Book b1 = new Book("Javaの基本", "山田太郎", 2800);
        Book b2 = new Book("デザインパターン", "鈴木花子", 3500);

        b1.show();
        b2.show();
        System.out.println(b1.title.equals(b2.title));
    }
}
▶ 答えを見る
▶ 出力結果
「Javaの基本」山田太郎 (2800円)
「デザインパターン」鈴木花子 (3500円)
false
解説
  • コンストラクタで各フィールドが初期化されます。
  • b1.title.equals(b2.title) は "Javaの基本".equals("デザインパターン") → false。
問題 7-2-2
出力を答える ⭐⭐☆

次のコードを実行したとき、出力される結果を答えてください。

Q2.java
class Temperature {
    double celsius;

    public Temperature(double celsius) {
        this.celsius = celsius;
    }

    public Temperature() {
        this(0.0);  // 引数ありコンストラクタに委譲
    }

    public double toFahrenheit() {
        return celsius * 9 / 5 + 32;
    }

    public void show() {
        System.out.printf("%.1f℃ = %.1f℉%n",
            celsius, toFahrenheit());
    }
}

public class Q2 {
    public static void main(String[] args) {
        Temperature t1 = new Temperature(100.0);
        Temperature t2 = new Temperature(0.0);
        Temperature t3 = new Temperature();

        t1.show();
        t2.show();
        t3.show();
        System.out.println(t1.celsius > t2.celsius);
    }
}
▶ 答えを見る
▶ 出力結果
100.0℃ = 212.0℉
0.0℃ = 32.0℉
0.0℃ = 32.0℉
true
解説
  • t1：100℃ → 100 × 9/5 + 32 = 212.0℉。
  • t2：0℃ → 0 × 9/5 + 32 = 32.0℉。
  • t3：引数なしコンストラクタが this(0.0) で celsius = 0.0 に設定されるため t2 と同じ出力。
  • t1.celsius > t2.celsius は 100.0 > 0.0 → true。
問題 7-2-3
コード記述 ⭐⭐⭐

Product（商品）クラスと Main クラスを作成してください。

要素 内容
フィールド String name、int price、int stock
static int totalProducts 生成したインスタンスの総数（クラス変数）
コンストラクタ1 (String, int, int) で全フィールドを初期化し totalProducts をインクリメント
コンストラクタ2 (String, int) で stock=0 としてコンストラクタ1に委譲
sell(int amount) 在庫が足りれば減らして "X個販売" を出力、足りなければ "在庫不足" を出力
show() "[name] 価格:X円 在庫:Y個" を出力
▶ 期待される出力
商品数: 3
[りんご] 価格:150円 在庫:100個
[バナナ] 価格:200円 在庫:50個
[メロン] 価格:800円 在庫:0個
5個販売
在庫不足
[りんご] 価格:150円 在庫:95個
▶ 答えを見る
Product.java + Main.java
class Product {
    String name;
    int    price;
    int    stock;
    static int totalProducts = 0;

    public Product(String name, int price, int stock) {
        this.name  = name;
        this.price = price;
        this.stock = stock;
        totalProducts++;
    }

    public Product(String name, int price) {
        this(name, price, 0);  // コンストラクタ1に委譲
    }

    public void sell(int amount) {
        if (amount > stock) {
            System.out.println("在庫不足");
        } else {
            stock -= amount;
            System.out.println(amount + "個販売");
        }
    }

    public void show() {
        System.out.println("[" + name + "] 価格:"
            + price + "円 在庫:" + stock + "個");
    }
}

public class Main {
    public static void main(String[] args) {
        Product p1 = new Product("りんご", 150, 100);
        Product p2 = new Product("バナナ", 200, 50);
        Product p3 = new Product("メロン", 800);     // stock=0

        System.out.println("商品数: " + Product.totalProducts);
        p1.show();
        p2.show();
        p3.show();

        p1.sell(5);
        p1.sell(200);   // 在庫不足
        p1.show();
    }
}
解説
  • totalProducts は static フィールドなので全インスタンスで共有され、コンストラクタが呼ばれるたびに増加します。
  • コンストラクタ2は this(name, price, 0) でコンストラクタ1に委譲し、処理を再利用します。
  • sell() は在庫チェック後に stock を更新します。
