終端操作とは？

Stream パイプラインを実際に実行して結果を得る操作です。
終端操作が呼ばれると Stream は消費され、再利用できません。

終端操作 説明
collect(Collector) List・Map・String など好きな形に収集
reduce(identity, BinaryOperator) 要素を累積して1つの値にまとめる
count() 要素数を返す（long）
forEach(Consumer) 各要素に処理を実行（戻り値なし）
findFirst() 最初の要素を Optional で返す
anyMatch / allMatch / noneMatch 条件に合う要素があるか確認
min / max 最小・最大値を Optional で返す
toArray() 配列に変換

collect() と Collectors

Collectors.toList()
stream.collect(Collectors.toList())
// → List<T>
Collectors.joining()
stream.collect(Collectors.joining(", "))
// → "a, b, c"
Collectors.groupingBy()
stream.collect(Collectors.groupingBy(f))
// → Map<K, List<T>>
Collectors.counting()
stream.collect(Collectors.counting())
// → Long（件数）
collect() のよく使う例
import java.util.stream.Collectors;

List<String> words = List.of("apple", "banana", "cherry", "avocado");

// toList
List<String> list = words.stream()
    .filter(w -> w.startsWith("a"))
    .collect(Collectors.toList());
System.out.println(list);  // [apple, avocado]

// joining
String joined = words.stream()
    .collect(Collectors.joining(", ", "[", "]"));
System.out.println(joined);  // [apple, banana, cherry, avocado]

// groupingBy（先頭文字でグループ化）
Map<Character, List<String>> grouped = words.stream()
    .collect(Collectors.groupingBy(w -> w.charAt(0)));
System.out.println(grouped);
// {a=[apple, avocado], b=[banana], c=[cherry]}

reduce() の使い方
List<Integer> nums = List.of(1, 2, 3, 4, 5);

// 合計（identity=0）
int sum = nums.stream()
    .reduce(0, (acc, n) -> acc + n);
System.out.println(sum);  // 15

// 積（identity=1）
int product = nums.stream()
    .reduce(1, (acc, n) -> acc * n);
System.out.println(product);  // 120

// 最大値（identity なし → Optional を返す）
Optional<Integer> max = nums.stream()
    .reduce((a, b) -> a > b ? a : b);
max.ifPresent(System.out::println);  // 5

// 文字列の連結
List<String> words = List.of("Hello", " ", "World", "!");
String sentence = words.stream()
    .reduce("", (acc, s) -> acc + s);
System.out.println(sentence);  // Hello World!

anyMatch / allMatch / noneMatch

条件チェックの終端操作
List<Integer> nums = List.of(2, 4, 6, 8, 9);

// anyMatch：1つでも条件を満たせば true
System.out.println(nums.stream().anyMatch(n -> n % 2 != 0));  // true（9がある）

// allMatch：全て条件を満たせば true
System.out.println(nums.stream().allMatch(n -> n % 2 == 0));  // false（9がある）

// noneMatch：全て条件を満たさなければ true
System.out.println(nums.stream().noneMatch(n -> n > 10));     // true

コード例：groupingBy の活用

CollectExample.java
import java.util.*;
import java.util.stream.*;

public class CollectExample {
    public static void main(String[] args) {
        List<String> words = List.of(
            "apple", "ant", "banana", "avocado",
            "cherry", "carrot", "blueberry"
        );

        // 先頭文字でグループ化 + 件数
        Map<Character, Long> countByInitial = words.stream()
            .collect(Collectors.groupingBy(
                w -> w.charAt(0),
                Collectors.counting()
            ));
        System.out.println(countByInitial);

        // 文字数でグループ化
        Map<Integer, List<String>> byLength = words.stream()
            .collect(Collectors.groupingBy(String::length));
        System.out.println(byLength);

        // 統計情報
        IntSummaryStatistics stats = words.stream()
            .mapToInt(String::length)
            .summaryStatistics();
        System.out.println("件数:" + stats.getCount()
            + " 合計:" + stats.getSum()
            + " 平均:" + stats.getAverage());
    }
}

Q1.java
import java.util.*;
import java.util.stream.*;

public class Q1 {
    public static void main(String[] args) {
        List<Integer> nums = List.of(3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5);

        // count
        long count = nums.stream().filter(n -> n >= 5).count();
        System.out.println("5以上の個数: " + count);

        // reduce で合計
        int sum = nums.stream().reduce(0, Integer::sum);
        System.out.println("合計: " + sum);

        // joining
        String joined = nums.stream()
            .distinct()
            .sorted()
            .map(String::valueOf)
            .collect(Collectors.joining("-"));
        System.out.println("ユニーク昇順: " + joined);

        // anyMatch / allMatch
        System.out.println(nums.stream().anyMatch(n -> n > 8));
        System.out.println(nums.stream().allMatch(n -> n > 0));
        System.out.println(nums.stream().noneMatch(n -> n < 0));
    }
}
▶ 答えを見る
▶ 出力結果
5以上の個数: 5
合計: 44
ユニーク昇順: 1-2-3-4-5-6-9
true
true
true
解説
  • 5以上：5(×3), 9(×1), 6(×1) = 5個。
  • 合計：3+1+4+1+5+9+2+6+5+3+5 = 44。
  • distinct() でユニーク化 → sorted() で昇順 → joining("-") で連結。
  • anyMatch(n → n > 8)：9があるので true。allMatch(n → n > 0)：全て正なので true。noneMatch(n → n < 0)：負の数なしなので true。
問題 11-2-2
出力を答える ⭐⭐⭐

次のコードを実行したとき、出力される結果を答えてください。

Q2.java
import java.util.*;
import java.util.stream.*;

public class Q2 {
    public static void main(String[] args) {
        List<String> words = List.of(
            "Java", "Stream", "Lambda", "API", "Filter", "Map", "Java"
        );

        // groupingBy で文字数グループ化（重複含む）
        Map<Integer, Long> countByLen = words.stream()
            .collect(Collectors.groupingBy(
                String::length,
                Collectors.counting()
            ));
        // TreeMap でキー昇順に出力
        new TreeMap<>(countByLen).forEach(
            (k, v) -> System.out.println(k + "文字: " + v + "件")
        );

        System.out.println("---");

        // joining でカンマ区切り（重複除去・ソート済み）
        String joined = words.stream()
            .distinct()
            .sorted()
            .collect(Collectors.joining(", "));
        System.out.println(joined);

        System.out.println("---");

        // reduce で最長の単語を求める
        Optional<String> longest = words.stream()
            .reduce((a, b) -> a.length() >= b.length() ? a : b);
        longest.ifPresent(s -> System.out.println("最長: " + s));
    }
}
▶ 答えを見る
▶ 出力結果
3文字: 2
4文字: 2
6文字: 2
7文字: 1
---
API, Filter, Java, Lambda, Map, Stream
---
最長: Stream
解説
  • 文字数グループ：API/Map(3文字×2)、Java(4文字×2・重複あり)、Filter/Stream(6文字×2)、Lambda(7文字×1)。
  • distinct() 後 sorted()：API, Filter, Java, Lambda, Map, Stream（アルファベット順）。
  • reduce で長い方を残していくと、最終的に最長の Stream(6) または Lambda(6) のうち先に現れた方が残ります。Stream と Lambda は同じ6文字ですが、a.length() >= b.length() のため最初に比較された方が保持され続けます → Stream が最後まで残ります。

次の条件を満たすクラスを作成してください。

  • Student クラス：name・score・grade（学年）フィールド
  • 以下の学生リストに対して Stream で集計処理を行う
名前 点数 学年

全員の合計点: 481

💡 ヒント：学年別平均は groupingBy + averagingInt を使います。
合格者名簿は点数70以上を filter し、名前を joining(", ") で連結します。
▶ 答えを見る
Main.java
import java.util.*;
import java.util.stream.*;

public class Main {

    static class Student {
        String name; int score; int grade;
        Student(String name, int score, int grade) {
            this.name=name; this.score=score; this.grade=grade;
        }
        String getName()  { return name; }
        int    getScore() { return score; }
        int    getGrade() { return grade; }
    }

    public static void main(String[] args) {
        List<Student> students = List.of(
            new Student("Alice", 85, 1),
            new Student("Bob",   72, 2),
            new Student("Carol", 91, 1),
            new Student("Dave",  60, 3),
            new Student("Eve",   78, 2),
            new Student("Frank", 95, 3)
        );

        // 合格者数
        long passed = students.stream()
            .filter(s -> s.getScore() >= 70)
            .count();
        System.out.println("合格者数（70点以上）: " + passed);

        // 合計点
        int total = students.stream()
            .mapToInt(Student::getScore)
            .sum();
        System.out.println("全員の合計点: " + total);

        // 平均点
        double avg = students.stream()
            .mapToInt(Student::getScore)
            .average()
            .orElse(0);
        System.out.println("平均点: " + avg);

        // 最高点の学生
        String topName = students.stream()
            .reduce((a, b) -> a.getScore() >= b.getScore() ? a : b)
            .map(Student::getName)
            .orElse("不明");
        System.out.println("最高点の学生: " + topName);

        // 学年別平均点
        Map<Integer, Double> avgByGrade = students.stream()
            .collect(Collectors.groupingBy(
                Student::getGrade,
                Collectors.averagingInt(Student::getScore)
            ));
        System.out.println("学年別平均点:");
        new TreeMap<>(avgByGrade).forEach(
            (g, a) -> System.out.println(g + "学年: " + a)
        );

        // 合格者名簿（点数順）
        String roster = students.stream()
            .filter(s -> s.getScore() >= 70)
            .sorted(Comparator.comparingInt(Student::getScore).reversed())
            .map(Student::getName)
            .collect(Collectors.joining(", "));
        System.out.println("合格者名簿: " + roster);
    }
}
解説
  • groupingBy + averagingInt で学年ごとの平均を Map<Integer,Double> として取得します。
  • reduce で点数が高い方を残し、map(Student::getName) で名前を取り出します。
  • 合格者名簿は filter → sorted（点数降順）→ map → joining のパイプラインで生成します。
