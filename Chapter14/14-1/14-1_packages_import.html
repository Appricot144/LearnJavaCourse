パッケージとは？

パッケージとは、関連するクラスを1つのフォルダにまとめる仕組みです。
ファイルシステムのディレクトリ構造と対応しています。

パッケージを使う主な理由は3つです。

  • 名前の衝突を防ぐ：異なるパッケージに同名のクラスを定義できる
  • コードを整理する：機能別にクラスをグループ化できる
  • アクセス制御：package-private（修飾子なし）でパッケージ内のみ公開できる

ディレクトリ構造との対応

パッケージ名はディレクトリ（フォルダ）の構造と一致します。

src/
  └─ com/
    └─ example/
      ├─ model/
      │   ├─ User.java ← package com.example.model;
      │   └─ Product.java
      ├─ service/
      │   └─ UserService.java ← package com.example.service;
      └─ util/
         └─ StringUtils.java ← package com.example.util;

package 宣言

ファイルの先頭に（コメントを除いて最初の文として）package 宣言を書きます。

User.java（com.example.model パッケージ）
package com.example.model;  // ← 先頭に宣言

public class User {
    private String name;
    private String email;

    public User(String name, String email) {
        this.name  = name;
        this.email = email;
    }

    public String getName()  { return name;  }
    public String getEmail() { return email; }
}

import 文

別パッケージのクラスを使うには import 文を書きます。

方法1：クラスを個別に import（推奨）
import java.util.ArrayList;
import java.util.List;
import java.util.HashMap;
方法2：パッケージ全体を import（* ワイルドカード） 
import java.util.*;  // java.util 全体
// ※サブパッケージは含まれない
方法3：完全修飾名（import なし）
java.util.ArrayList<String> list = new java.util.ArrayList<>();
// 毎回フルパスを書く（冗長だが import 不要）
方法4：static import（静的メンバーを直接使う） 
import static java.lang.Math.*;  // Math の全 static メンバー
import static java.util.Collections.sort;

double r = sqrt(16.0);  // Math.sqrt() を省略して使える
💡 java.lang パッケージは自動 import
String・Integer・Math・System など java.lang パッケージのクラスは
import 不要で使えます。Javaが自動的に import しているためです。

よく使う import 一覧

import 文 使えるクラス例
import java.util.*; List、Map、HashMap、ArrayList、Optional
import java.util.stream.*; Stream、Collectors、IntStream
import java.time.*; LocalDate、LocalDateTime、Period
import java.time.format.*; DateTimeFormatter
import java.util.function.*; Function、Predicate、Consumer
import java.io.*; File、IOException、BufferedReader

名前の衝突と完全修飾名

同じクラス名が異なるパッケージに存在する場合、片方を完全修飾名で書きます。

java.util.Date と java.sql.Date の衝突例
import java.util.Date;  // java.util.Date は import

// java.sql.Date は完全修飾名で使う
java.sql.Date sqlDate = new java.sql.Date(System.currentTimeMillis());
Date utilDate = new Date();  // こちらは import 済みなので短い名前で OK

コード例：自前パッケージの使い方

com/example/model/Product.java
package com.example.model;

public class Product {
    private String name;
    private int    price;

    public Product(String name, int price) {
        this.name  = name;
        this.price = price;
    }
    public String getName()  { return name;  }
    public int    getPrice() { return price; }

    @Override
    public String toString() {
        return name + "(" + price + "円)";
    }
}
com/example/Main.java
package com.example;

import com.example.model.Product;  // 同じプロジェクトの別パッケージ
import java.util.List;
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        List<Product> cart = new ArrayList<>();
        cart.add(new Product("りんご",  150));
        cart.add(new Product("バナナ",  120));
        cart.add(new Product("メロン", 1200));

        int total = cart.stream()
            .mapToInt(Product::getPrice)
            .sum();

        System.out.println("合計: " + total + "円");
    }
}
▶ 出力結果
合計: 1470円

問題 14-1-1
選択問題 ⭐☆☆

次の各文について、正しいものには ✅、誤りには ❌ を答えてください。

  1. package 宣言はファイルのどこに書いても良い。
  2. import java.util.*; は java.util.stream パッケージのクラスも import する。
  3. java.lang.String は import なしで使える。
  4. 異なるパッケージに同名のクラスを定義することができる。
  5. import static java.lang.Math.sqrt; と書くと sqrt(4.0) と省略して呼べる。
▶ 答えを見る
1. ❌：package 宣言はファイルの先頭（コメントを除く最初の文）に書かなければなりません。
2. ❌：import java.util.*; は java.util パッケージのクラスのみ対象です。java.util.stream のようなサブパッケージは含まれません。別途 import java.util.stream.*; が必要です。
3. ✅：java.lang パッケージは Java が自動的に import するため、String・Math・Integer などは import 不要です。
4. ✅：パッケージが異なれば同名クラスを定義できます（完全修飾名が異なるため）。
5. ✅：static import でメソッド名のみで呼び出せます。
問題 14-1-2
出力を答える ⭐⭐☆

import java.util.ArrayList;
import java.util.List;
import java.util.Collections;
import static java.lang.Math.max;
import static java.lang.Math.min;
import static java.lang.Math.abs;

public class Q2 {
    public static void main(String[] args) {
        // static import で Math.xxx() を省略
        System.out.println(max(10, 20));
        System.out.println(min(10, 20));
        System.out.println(abs(-42));

        List<Integer> nums = new ArrayList<>();
        nums.add(5); nums.add(2); nums.add(8); nums.add(1);

        Collections.sort(nums);
        System.out.println(nums);

        Collections.reverse(nums);
        System.out.println(nums);

        System.out.println(Collections.max(nums));
        System.out.println(Collections.min(nums));
    }
}
▶ 答えを見る

解説
  • static import で Math.max/min/abs を短縮名で使っています。
  • Collections.sort() で昇順ソート → [1, 2, 5, 8]。
  • Collections.reverse() で逆順 → [8, 5, 2, 1]。
問題 14-1-3
コード記述 ⭐⭐⭐

次の条件を満たすクラス群を作成してください。

  • com.example.shape パッケージに Shape 抽象クラスを定義（color フィールド・abstract area()・show()）
  • 同パッケージに Circle と Rectangle クラスを定義（Shape を継承・area() を実装）
  • com.example パッケージの Main クラスで import com.example.shape.*; を使って利用する
▶ 期待される出力
Circle（赤）面積: 78.54
Rectangle（青）面積: 24.00
Circle（緑）面積: 28.27
合計面積: 130.81
▶ 答えを見る
com/example/shape/Shape.java
package com.example.shape;

public abstract class Shape {
    protected String color;

    public Shape(String color) { this.color = color; }

    public abstract double area();

    public void show() {
        System.out.printf("%s（%s）面積: %.2f%n",
            getClass().getSimpleName(), color, area());
    }
}
com/example/shape/Circle.java
package com.example.shape;

public class Circle extends Shape {
    private double radius;

    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }

    @Override
    public double area() { return Math.PI * radius * radius; }
}
com/example/shape/Rectangle.java
package com.example.shape;

public class Rectangle extends Shape {
    private double width, height;

    public Rectangle(String color, double width, double height) {
        super(color);
        this.width = width; this.height = height;
    }

    @Override
    public double area() { return width * height; }
}
com/example/Main.java
package com.example;

import com.example.shape.*;  // Shape, Circle, Rectangle を一括 import

public class Main {
    public static void main(String[] args) {
        Shape[] shapes = {
            new Circle("赤", 5.0),
            new Rectangle("青", 4.0, 6.0),
            new Circle("緑", 3.0)
        };

        double total = 0;
        for (Shape s : shapes) {
            s.show();
            total += s.area();
        }
        System.out.printf("合計面積: %.2f%n", total);
    }
}
解説
  • package com.example.shape; で各クラスが同パッケージに属することを宣言します。
  • import com.example.shape.*; でパッケージ全体のクラスを使えるようにします。
  • 抽象クラス Shape を参照型として使い、Circle と Rectangle のインスタンスをポリモーフィズムで一括処理します。
