コード例：標準ライブラリを組み合わせる

StandardLibExample.java
import java.util.*;
import java.util.stream.*;

public class StandardLibExample {
    public static void main(String[] args) {
        // Math + Stream
        List<Double> values = List.of(1.5, -2.3, 4.0, -0.5, 3.7);

        double sumAbs = values.stream()
            .mapToDouble(Math::abs)
            .sum();
        System.out.printf("絶対値の合計: %.1f%n", sumAbs);  // 12.0

        // StringBuilder で表組み
        List<String> headers = List.of("名前", "点数", "評価");
        StringBuilder table = new StringBuilder();
        table.append(String.join(" | ", headers)).append("\n");
        table.append("-".repeat(20)).append("\n");
        table.append(String.join(" | ",
            List.of("Alice", "85", "A"))).append("\n");
        table.append(String.join(" | ",
            List.of("Bob",   "72", "B"))).append("\n");
        System.out.println(table);

        // Collections.frequency で出現回数
        List<String> words = List.of("a","b","a","c","a","b");
        System.out.println("a の出現数: "
            + Collections.frequency(words, "a"));  // 3
    }
}

問題 14-2-3
コード記述 ⭐⭐⭐

次の条件を満たすクラスを作成してください。

  • Math クラスを使って数学的な処理を行う
  • StringBuilder で成績表を組み立てる
  • Collections を使ってランキングを作成する
処理 内容
偏差値計算 点数リスト {85, 72, 91, 60, 78} の平均・標準偏差・各点数の偏差値を計算
成績表組み立て StringBuilder で名前と偏差値のテーブルを作成
ランキング 点数を降順にソートして出力
▶ 期待される出力
平均: 77.2
標準偏差: 10.69
=== 偏差値 ===
Alice : 57.8
Bob   : 47.2
Carol : 63.6
Dave  : 37.2
Eve   : 50.7
---
降順ランキング: [91, 85, 78, 72, 60]
💡 ヒント：偏差値 = (点数 - 平均) / 標準偏差 × 10 + 50
標準偏差 = Math.sqrt(各点数の (点数-平均)² の平均)
▶ 答えを見る
Main.java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        String[] names  = {"Alice", "Bob", "Carol", "Dave", "Eve"};
        int[]    scores = {85, 72, 91, 60, 78};

        // 平均
        double sum = 0;
        for (int s : scores) sum += s;
        double avg = sum / scores.length;
        System.out.printf("平均: %.1f%n", avg);

        // 標準偏差
        double variance = 0;
        for (int s : scores) variance += Math.pow(s - avg, 2);
        double stdDev = Math.sqrt(variance / scores.length);
        System.out.printf("標準偏差: %.2f%n", stdDev);

        // StringBuilder で成績表
        StringBuilder sb = new StringBuilder("=== 偏差値 ===\n");
        for (int i = 0; i < scores.length; i++) {
            double deviation = (scores[i] - avg) / stdDev * 10 + 50;
            sb.append(String.format("%-6s: %.1f%n", names[i], deviation));
        }
        System.out.print(sb);

        // Collections で降順ランキング
        List<Integer> ranking = new ArrayList<>();
        for (int s : scores) ranking.add(s);
        Collections.sort(ranking, Comparator.reverseOrder());
        System.out.println("---\n降順ランキング: " + ranking);
    }
}
解説
  • 平均：(85+72+91+60+78)/5 = 77.2。
  • 分散：各(x-77.2)²の平均。標準偏差≒10.69。
  • 偏差値：Alice = (85-77.2)/10.69×10+50 ≈ 57.3（実際の計算値に依存）。
  • String.format("%-6s", name)で左揃え6文字幅にパディング。
