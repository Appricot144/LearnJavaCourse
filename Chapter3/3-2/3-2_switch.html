switch文とは？

switch 文は、1つの変数や式の値によって処理を分岐させる構文です。
if / else if の連続と同じことができますが、値が多い場合に読みやすくなります。

基本構造

switch文の構文
switch (式) {
    case 値1:
        // 式 == 値1 のときの処理
        break;
    case 値2:
        // 式 == 値2 のときの処理
        break;
    default:
        // どの case にも一致しないときの処理
}
⚠️ break を忘れずに！
break を省略すると次の case に処理が流れ続けます（フォールスルー）。
意図しない動作の原因になるため、必ず記述する習慣をつけましょう。

break を省略するとどうなる？（フォールスルー）

フォールスルーの例（x = 2）
int x = 2;

switch (x) {
    case 1:
        System.out.println("A");
        break;
    case 2:
        System.out.println("B");  // ← 実行される
        // break なし！
    case 3:
        System.out.println("C");  // ← フォールスルーで実行される
        break;
    case 4:
        System.out.println("D");
        break;
}

switch文で使える型

使える型 例
int / byte / short / char switch (score)
String switch (name)
enum（列挙型） switch (color)
⚠️ 使えない型：long / float / double / boolean は switch 文では使えません。

switch文 vs if文

比較項目 switch文 if文
向いているケース 値が多い・1変数の等値比較 範囲比較・複雑な条件
使える比較 ==（等値のみ） >=、&& など自由
読みやすさ 値が多い場合に○ 条件が複雑な場合に○

コード例（String の switch）

SeasonChecker.java
public class SeasonChecker {
    public static void main(String[] args) {
        String season = "summer";

        switch (season) {
            case "spring":
                System.out.println("春：お花見の季節");
                break;
            case "summer":
                System.out.println("夏：海水浴の季節");  // ← これが実行される
                break;
            case "autumn":
                System.out.println("秋：紅葉の季節");
                break;
            case "winter":
                System.out.println("冬：スキーの季節");
                break;
            default:
                System.out.println("不明な季節");
        }
    }
}
▶ 出力結果
夏：海水浴の季節

問題 3-2-3
コード記述 ⭐⭐⭐

次の条件を満たす TrafficLight クラスを作成してください。

  • String 型の変数 color に "yellow" を代入する
  • switch 文を使って下表のメッセージを出力する
  • "red" または "yellow" のとき、追加で "安全を確認してください" を出力する
color の値 出力メッセージ
"red" 止まれ
"yellow" 注意して進め
"green" 進め
それ以外 不明な信号です
▶ 期待される出力
注意して進め
安全を確認してください
▶ 答えを見る
TrafficLight.java
public class TrafficLight {
    public static void main(String[] args) {
        String color = "yellow";

        switch (color) {
            case "red":
                System.out.println("止まれ");
                System.out.println("安全を確認してください");
                break;
            case "yellow":
                System.out.println("注意して進め");
                System.out.println("安全を確認してください");
                break;
            case "green":
                System.out.println("進め");
                break;
            default:
                System.out.println("不明な信号です");
        }
    }
}
▶ 別解（共通処理を switch の外にまとめる）
TrafficLight.java（別解）
public class TrafficLight {
    public static void main(String[] args) {
        String color = "yellow";
        String message = "";
        boolean needsCheck = false;

        switch (color) {
            case "red":
                message = "止まれ";
                needsCheck = true;
                break;
            case "yellow":
                message = "注意して進め";
                needsCheck = true;
                break;
            case "green":
                message = "進め";
                break;
            default:
                message = "不明な信号です";
        }

        System.out.println(message);
        if (needsCheck) {
            System.out.println("安全を確認してください");
        }
    }
}
解説
  • color = "yellow" なので case "yellow": に一致します。
  • "注意して進め" と "安全を確認してください" が出力されます。
  • 別解では red と yellow の共通処理（安全確認）を switch の外で if にまとめています。コードの重複を減らせるメリットがあります。
