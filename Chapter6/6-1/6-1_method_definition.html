メソッドとは？

一連の処理をまとめて名前を付けたものです。
同じ処理を何度も書く代わりに、メソッドとして定義して呼び出します。

♻️
再利用性
同じ処理を何度でも呼び出せる
📖
可読性
処理に名前が付いて意図が明確に
🔧
保守性
変更箇所が1カ所にまとまる

メソッドの呼び出し

定義と呼び出しの例
public class MethodExample {

    // メソッドの定義
    public static void greet() {
        System.out.println("こんにちは！");
    }

    public static void main(String[] args) {
        greet();  // 呼び出し1回目
        greet();  // 呼び出し2回目
        greet();  // 呼び出し3回目
    }
}
▶ 出力結果
こんにちは！
こんにちは！
こんにちは！

メソッドの実行フロー

▶ main から greet を呼び出す流れ
main() {
  ① greet(); ──→
  ③ greet(); ──→
}
⇄
greet() {
  ② println("こんにちは！")
  ← mainに戻る
}

void メソッドと return

void メソッドは戻り値がありませんが、return; で途中終了できます。

途中終了の例
public static void checkAge(int age) {
    if (age < 0) {
        System.out.println("無効な年齢です");
        return;  // ここで終了
    }
    System.out.println("年齢: " + age);
}

メソッドの命名規則

ルール例
小文字で始めるキャメルケースcalculateTotal、getUserName
動詞または動詞＋名詞print、getScore、showResult
略語は避けるcalculate ○、calc △

public class ShapeDrawer {

問題 6-1-1
出力を答える ⭐☆☆

次のコードを実行したとき、出力される結果を答えてください。

Q1.java
public class Q1 {

    public static void hello() {
        System.out.println("Hello!");
    }

    public static void bye() {
        System.out.println("Goodbye!");
    }

    public static void greetSet() {
        hello();
        bye();
    }

    public static void main(String[] args) {
        hello();
        greetSet();
        bye();
    }
}
▶ 答えを見る
▶ 出力結果
Hello!
Hello!
Goodbye!
Goodbye!
解説
  • main の hello() → "Hello!"
  • greetSet() が呼ばれ、内部で hello() → "Hello!"、bye() → "Goodbye!"
  • main の bye() → "Goodbye!"

出力を答える ⭐⭐☆

Q2.java
public class Q2 {

    public static void printStars(int count) {
        for (int i = 0; i < count; i++) {
            System.out.print("*");
        }
        System.out.println();
    }

    public static void printTriangle() {
        printStars(1);
        printStars(2);
        printStars(3);
        printStars(4);
        printStars(5);
    }

    public static void main(String[] args) {
        printTriangle();
        System.out.println("---");
        printStars(3);
    }
}
▶ 答えを見る
▶ 出力結果
*
**
***
****
*****
---
***
解説
  • printTriangle() は printStars(1)〜printStars(5) を順に呼び出し1〜5個の * を出力します。
  • System.out.println("---") でセパレータを出力。
  • printStars(3) で *** を出力します。

ReportPrinter.java
public class ReportPrinter {

    public static void printHeader() {
        System.out.println("=== 成績レポート ===");
    }

    public static void printSeparator() {
        System.out.println("-------------------");
    }

    public static void printFooter() {
        System.out.println("=== 以上 ===");
    }

    public static void main(String[] args) {
        printHeader();
        printSeparator();
        System.out.println("名前: 山田太郎");
        System.out.println("点数: 85点");
        System.out.println("評価: B");
        printSeparator();
        System.out.println("名前: 佐藤花子");
        System.out.println("点数: 92点");
        System.out.println("評価: A");
        printSeparator();
        printFooter();
    }
}
解説
  • 繰り返し使うセパレータ行をメソッドにまとめることで、変更が1カ所で済みます。
  • printSeparator() は3回呼ばれていますが、定義は1カ所です。これがメソッドの再利用性の利点です。
