Enum とは？

あらかじめ決まった定数の集まりを型として定義する仕組みです。
曜日・方角・状態など「選択肢が固定されているもの」に使います。

❌ Enum なし（バグの元）
int day = 8;  // 8は何曜日？
// 7を超えてもエラーにならない
✅ Enum あり（型安全）
Day day = Day.MONDAY;
// Day型以外は代入できない

基本的な Enum の定義

▶ Day Enum の各定数（ordinal は 0 始まり）
MONDAY0
TUESDAY1
WEDNESDAY2
THURSDAY3
FRIDAY4
SATURDAY5
SUNDAY6
基本的な Enum と使い方
public enum Day {
    MONDAY, TUESDAY, WEDNESDAY, THURSDAY,
    FRIDAY, SATURDAY, SUNDAY
}

Day today = Day.WEDNESDAY;
System.out.println(today);            // WEDNESDAY
System.out.println(today.name());     // WEDNESDAY（文字列）
System.out.println(today.ordinal());  // 2（0始まりの番号）

// switch 文と組み合わせる
switch (today) {
    case SATURDAY: case SUNDAY:
        System.out.println("週末！"); break;
    default:
        System.out.println("平日");
}

// 比較は == を使う（Enum は == で比較できる）
if (today == Day.WEDNESDAY) System.out.println("水曜日です");

フィールドとメソッドを持つ Enum

Status Enum の構造
定数
PENDING("受付中"), PROCESSING("処理中"), COMPLETED("完了"), CANCELLED("キャンセル")
フィールド
private final String label;
メソッド
getLabel() → ラベル文字列を返す
isFinished() → COMPLETED または CANCELLED なら true
Status Enum の実装
public enum Status {
    PENDING("受付中"), PROCESSING("処理中"),
    COMPLETED("完了"), CANCELLED("キャンセル");

    private final String label;

    Status(String label) { this.label = label; }

    public String getLabel() { return label; }

    public boolean isFinished() {
        return this == COMPLETED || this == CANCELLED;
    }
}

Status s = Status.PROCESSING;
System.out.println(s.getLabel());    // 処理中
System.out.println(s.isFinished());  // false

for (Status st : Status.values()) {
    System.out.println(st.ordinal() + ": " + st.getLabel());
}
▶ 出力結果
処理中
false
0: 受付中
1: 処理中
2: 完了
3: キャンセル

Enum のよく使うメソッド

メソッド 説明
name() Enum 定数の名前を文字列で返す
ordinal() Enum 定数の番号（0始まり）を返す
values() すべての定数を配列で返す
valueOf(String) 文字列から Enum 定数を取得する
compareTo(other) ordinal() の差を返す

コード例：信号機シミュレーション

TrafficLight Enum
public enum TrafficLight {
    RED("止まれ", 30), YELLOW("注意", 5), GREEN("進め", 25);

    private final String message;
    private final int    seconds;

    TrafficLight(String message, int seconds) {
        this.message = message; this.seconds = seconds;
    }

    public String getMessage() { return message; }
    public int    getSeconds() { return seconds; }

    public TrafficLight next() {
        TrafficLight[] values = values();
        return values[(ordinal() + 1) % values.length];
    }
}

// Main
TrafficLight light = TrafficLight.RED;
for (int i = 0; i < 4; i++) {
    System.out.println(light + ": " + light.getMessage()
        + "（" + light.getSeconds() + "秒）");
    light = light.next();
}
▶ 出力結果
RED: 止まれ（30秒）
YELLOW: 注意（5秒）
GREEN: 進め（25秒）
RED: 止まれ（30秒）

enum Season {
    SPRING("春", 3), SUMMER("夏", 6),
    AUTUMN("秋", 9), WINTER("冬", 12);

    final String name;
    final int    startMonth;

    Season(String name, int startMonth) {
        this.name = name; this.startMonth = startMonth;
    }

    boolean isWarm() { return this == SPRING || this == SUMMER; }
}

public class Q1 {
    public static void main(String[] args) {
        Season s = Season.SUMMER;
        System.out.println(s);
        System.out.println(s.name);
        System.out.println(s.ordinal());
        System.out.println(s.startMonth);
        System.out.println(s.isWarm());

        System.out.println("---");

        for (Season season : Season.values()) {
            System.out.println(season.name + ": " + season.isWarm());
        }
    }
}

▶ 出力結果

夏

true
---

夏: true
秋: false
冬: false
解説
  • s.toString() は Enum のデフォルトで "SUMMER"。
  • s.name は独自フィールドの "夏"（s.name() メソッドとは別物）。
  • s.ordinal() は SPRING=0, SUMMER=1 なので 1。
  • isWarm() は SPRING と SUMMER のみ true。
問題 9-3-2

Q2.java
enum Priority {
    LOW(1), MEDIUM(5), HIGH(10), CRITICAL(100);

    final int weight;
    Priority(int weight) { this.weight = weight; }
    boolean isUrgent() { return weight >= 10; }
}

class Task {
    String title; Priority priority;

    public Task(String title, Priority priority) {
        this.title = title; this.priority = priority;
    }

    public void show() {
        System.out.println("[" + priority + "(" + priority.weight + ")] "
            + title + (priority.isUrgent() ? " ⚠" : ""));
    }
}

public class Q2 {
    public static void main(String[] args) {
        Task[] tasks = {
            new Task("バグ修正",       Priority.CRITICAL),
            new Task("ドキュメント",   Priority.LOW),
            new Task("テスト追加",     Priority.HIGH),
            new Task("コードレビュー", Priority.MEDIUM)
        };

        for (Task t : tasks) t.show();

        System.out.println("---");
        System.out.println(Priority.HIGH.compareTo(Priority.LOW));
        System.out.println(Priority.valueOf("MEDIUM").weight);
    }
}
▶ 答えを見る
▶ 出力結果
[CRITICAL(100)] バグ修正 ⚠
[LOW(1)] ドキュメント
[HIGH(10)] テスト追加 ⚠
[MEDIUM(5)] コードレビュー
---
2
5
解説
  • isUrgent() は weight >= 10 なので HIGH と CRITICAL が true → " ⚠" が付きます。
  • Priority.HIGH.compareTo(Priority.LOW) は ordinal の差 = 2 - 0 = 2。
  • Priority.valueOf("MEDIUM").weight → Priority.MEDIUM.weight = 5。

Coin Enum と Main クラスを作成してください。

定数 金額（yen）

YEN_5 5
YEN_10 10
YEN_50 50
YEN_100 100
YEN_500 500
  • int yen フィールドと getYen() getter を持つ
  • static int calcTotal(Coin[] coins) で配列の合計金額を返す
▶ 期待される出力
全コイン一覧:
1円 / 5円 / 10円 / 50円 / 100円 / 500円 / 
---
コインの合計: 666円
最大コイン: YEN_500（500円）
💡 ヒント：wallet = {YEN_500, YEN_100, YEN_50, YEN_10, YEN_5, YEN_1} を使います。
合計は 500+100+50+10+5+1 = 666。最大コインは wallet 配列から探します。
▶ 答えを見る
Coin.java + Main.java
enum Coin {
    YEN_1(1), YEN_5(5), YEN_10(10),
    YEN_50(50), YEN_100(100), YEN_500(500);

    private final int yen;
    Coin(int yen) { this.yen = yen; }
    public int getYen() { return yen; }

    public static int calcTotal(Coin[] coins) {
        int total = 0;
        for (Coin c : coins) total += c.yen;
        return total;
    }
}

public class Main {
    public static void main(String[] args) {
        System.out.println("全コイン一覧:");
        for (Coin c : Coin.values()) {
            System.out.print(c.getYen() + "円 / ");
        }
        System.out.println();
        System.out.println("---");

        Coin[] wallet = {
            Coin.YEN_500, Coin.YEN_100, Coin.YEN_50,
            Coin.YEN_10,  Coin.YEN_5,   Coin.YEN_1
        };

        System.out.println("コインの合計: " + Coin.calcTotal(wallet) + "円");

        Coin max = wallet[0];
        for (Coin c : wallet) {
            if (c.getYen() > max.getYen()) max = c;
        }
        System.out.println("最大コイン: " + max + "（" + max.getYen() + "円）");
    }
}
解説
  • Coin.values() で全定数を順に取得して出力します。
  • calcTotal は配列の各 yen を合計します（1+5+10+50+100+500=666）。
  • 最大コインは wallet 配列から getYen() で比較して求めます。
