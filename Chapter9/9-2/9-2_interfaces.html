インターフェースとは？

メソッドのシグネチャだけを定義した「契約」です。
クラスが implements することで、そのメソッドの実装を保証します。

▶ Flyable インターフェースを Bird と Airplane が実装する
《interface》
Flyable
void fly()
int getMaxAltitude()
↓ implements
Bird
fly() → "羽ばたいて飛ぶ"
getMaxAltitude() → 500
Airplane
fly() → "エンジンで飛ぶ"
getMaxAltitude() → 12000

インターフェースの定義

Flyable インターフェース
public interface Flyable {
    void fly();                  // 抽象メソッド（abstract は省略可）
    int  getMaxAltitude();

    // Java 8以降：default メソッド（実装あり）
    default String status() { return "飛行中"; }
}

implements による実装

Bird と Airplane の実装
public class Bird implements Flyable {
    String name;
    public Bird(String name) { this.name = name; }

    @Override public void fly() { System.out.println(name+" が羽ばたいて飛ぶ"); }
    @Override public int  getMaxAltitude() { return 500; }
}

public class Airplane implements Flyable {
    String model;
    public Airplane(String model) { this.model = model; }

    @Override public void fly() { System.out.println(model+" がエンジンで飛ぶ"); }
    @Override public int  getMaxAltitude() { return 12000; }
}

継承との最大の違い：複数実装

▶ Duck は Flyable と Swimmable を両方実装できる
Flyable
+
Swimmable
→
Duck implements Flyable, Swimmable
複数インターフェースの実装
public interface Swimmable { void swim(); }
public interface Flyable   { void fly(); }

public class Duck implements Flyable, Swimmable {
    @Override public void fly()  { System.out.println("カモが飛ぶ"); }
    @Override public void swim() { System.out.println("カモが泳ぐ"); }
}

インターフェース型による参照

Flyable 型で統一的に扱う
Flyable[] flyers = { new Bird("ハト"), new Airplane("A380") };

for (Flyable f : flyers) {
    f.fly();
    System.out.println("最高高度: " + f.getMaxAltitude() + "m");
}
// → ハト が羽ばたいて飛ぶ
// → 最高高度: 500m
// → A380 がエンジンで飛ぶ
// → 最高高度: 12000m

抽象クラス vs インターフェース

比較 抽象クラス インターフェース
目的 共通の「骨格」 「〜できる」能力の保証
フィールド 自由に定義可 public static final のみ
コンストラクタ 持てる 持てない
多重継承 1つだけ 複数実装可能
向いているケース is-a 関係（犬は動物） can-do 関係（鳥は飛べる）

コード例：Printable と Saveable

Report クラスが2つのインターフェースを実装
interface Printable { void print(); }
interface Saveable  { void save(String filename); }

class Report implements Printable, Saveable {
    String content;
    public Report(String content) { this.content = content; }

    @Override public void print() {
        System.out.println("=== 印刷 ===\n" + content);
    }
    @Override public void save(String filename) {
        System.out.println(filename + " に保存しました：" + content);
    }
}

// Main
Report r = new Report("売上レポート");
r.print();
r.save("report.txt");
▶ 出力結果
=== 印刷 ===
売上レポート
report.txt に保存しました：売上レポート

問題 9-2-1
出力を答える ⭐⭐☆

次のコードを実行したとき、出力される結果を答えてください。

Q1.java
interface Greetable {
    void greet();
    default String farewell() { return "さようなら"; }
}

interface Introducible {
    String introduce();
}

class EnglishSpeaker implements Greetable, Introducible {
    String name;
    public EnglishSpeaker(String name) { this.name = name; }

    @Override public void greet()       { System.out.println("Hello! I'm " + name); }
    @Override public String introduce() { return "My name is " + name + "."; }
}

class JapaneseSpeaker implements Greetable {
    String name;
    public JapaneseSpeaker(String name) { this.name = name; }
    @Override public void greet() { System.out.println("こんにちは、" + name + "です"); }
}

public class Q1 {
    public static void main(String[] args) {
        EnglishSpeaker  e = new EnglishSpeaker("Alice");
        JapaneseSpeaker j = new JapaneseSpeaker("田中");

        e.greet();
        System.out.println(e.introduce());
        System.out.println(e.farewell());

        j.greet();
        System.out.println(j.farewell());

        Greetable[] speakers = { e, j };
        for (Greetable g : speakers) g.greet();
    }
}
▶ 答えを見る
▶ 出力結果
Hello! I'm Alice
My name is Alice.
さようなら
こんにちは、田中です
さようなら
Hello! I'm Alice
こんにちは、田中です
解説
  • farewell() は Greetable の default メソッドなので、実装クラスで上書きしなければそのまま使われます。
  • Greetable[] に2クラスを混在させ、greet() を統一的に呼び出しています。
  • introduce() は Greetable にはないため、Greetable 型では呼べません。
問題 9-2-2
出力を答える ⭐⭐⭐

次のコードを実行したとき、出力される結果を答えてください。

Q2.java
interface Comparable2 {
    int compareTo(int other);
    default boolean isGreaterThan(int other) { return compareTo(other) > 0; }
    default boolean isLessThan(int other)    { return compareTo(other) < 0; }
}

class Score implements Comparable2 {
    int value;
    public Score(int value) { this.value = value; }

    @Override public int compareTo(int other) { return this.value - other; }
    @Override public String toString() { return "Score(" + value + ")"; }
}

public class Q2 {
    public static void main(String[] args) {
        Score s1 = new Score(85);
        Score s2 = new Score(60);

        System.out.println(s1.compareTo(60));
        System.out.println(s1.isGreaterThan(60));
        System.out.println(s2.isLessThan(70));
        System.out.println(s1.isGreaterThan(90));

        Comparable2 c = s1;
        System.out.println(c.isGreaterThan(80));
    }
}
▶ 答えを見る
▶ 出力結果
25
true
true
false
true
解説
  • s1.compareTo(60) → 85 - 60 = 25。
  • s1.isGreaterThan(60) → 25 > 0 → true。
  • s2.isLessThan(70) → 60 - 70 = -10 < 0 → true。
  • s1.isGreaterThan(90) → 85 - 90 = -5 > 0 → false。
  • Comparable2 c = s1 でも動的ディスパッチで Score.compareTo() が呼ばれます。
問題 9-2-3
コード記述 ⭐⭐⭐

Taxable・Discountable インターフェースと SaleItem クラスを作成してください。

要素 内容
Taxable getTaxRate() 抽象メソッド。default getTax(price)・default getTotalPrice(price)
Discountable getDiscountRate() 抽象メソッド。default getDiscountedPrice(price)
SaleItem 両インターフェースを実装。getTaxRate()=0.10・getDiscountRate()=0.20。showPrice() で価格詳細を出力
▶ 期待される出力
商品: りんご
税抜元値: 1000.0円
割引後: 800.0円
税込合計: 880.0円
💡 ヒント：showPrice() では getDiscountedPrice(basePrice) を先に計算し、
その結果に getTotalPrice(discounted) を適用します。
▶ 答えを見る
Taxable / Discountable / SaleItem + Main 
interface Taxable {
    double getTaxRate();
    default double getTax(double price)        { return price * getTaxRate(); }
    default double getTotalPrice(double price) { return price + getTax(price); }
}

interface Discountable {
    double getDiscountRate();
    default double getDiscountedPrice(double price) {
        return price * (1 - getDiscountRate());
    }
}

class SaleItem implements Taxable, Discountable {
    String name;
    double basePrice;

    public SaleItem(String name, double basePrice) {
        this.name = name; this.basePrice = basePrice;
    }

    @Override public double getTaxRate()      { return 0.10; }
    @Override public double getDiscountRate() { return 0.20; }

    public void showPrice() {
        double discounted = getDiscountedPrice(basePrice);
        double total      = getTotalPrice(discounted);
        System.out.println("商品: "     + name);
        System.out.println("税抜元値: " + basePrice + "円");
        System.out.println("割引後: "   + discounted + "円");
        System.out.println("税込合計: " + total + "円");
    }
}

public class Main {
    public static void main(String[] args) {
        new SaleItem("りんご", 1000.0).showPrice();
    }
}
解説
  • getDiscountedPrice(1000.0) → 1000.0 × 0.8 = 800.0。
  • getTotalPrice(800.0) → 800.0 + 80.0 = 880.0。
  • SaleItem は Taxable と Discountable 両方の default メソッドを利用します。
